# 2286. Booking Concert Tickets in Groups

## O(log n) per operation — Segment Tree

---

## Problem Statement

- Concert hall: `n` rows × `m` seats per row.
- **Gather(k, maxRow)**: allocate `k` consecutive seats in same row ≤ maxRow. Return [row, startSeat] or [].
- **Scatter(k, maxRow)**: allocate `k` seats (may be non-consecutive) in rows ≤ maxRow. Return true/false.
- **Constraints**: 1 ≤ n ≤ 5×10⁴, 1 ≤ m, k ≤ 10⁹, up to 5×10⁴ operations

---

## Core Idea — Segment Tree for Range Queries

**Key operations needed**:
1. **Find first row** with ≥ k consecutive empty seats (for gather)
2. **Sum empty seats** across range of rows (for scatter)
3. **Update** seat counts efficiently

**Data structure**: Segment tree maintaining:
- `maxFree[node]`: max consecutive free seats in range
- `totalFree[node]`: total free seats in range

---

## Algorithm

### Segment Tree Structure

```
Node represents range [left, right] of rows:
  - maxFree: max free consecutive seats in any row in range
  - totalFree: sum of free seats across all rows in range
```

### Gather Operation

```
Gather(k, maxRow):
  1. Query segment tree for first row in [0, maxRow] with maxFree >= k
  2. If found at row r:
     - Allocate seats starting at position (m - free[r])
     - Update free[r] -= k
     - Update segment tree
     - Return [r, startSeat]
  3. Else return []
```

### Scatter Operation

```
Scatter(k, maxRow):
  1. Query totalFree in range [0, maxRow]
  2. If totalFree < k: return false
  3. Allocate k seats greedily from row 0 upward:
     - Take min(k, free[row]) from each row
     - Update segment tree
  4. Return true
```

## Full Implementation (C#)

```csharp
public class BookMyShow {
    private int n, m;
    private long[] maxFree; // Segment tree: max free in range
    private long[] totalFree; // Segment tree: total free in range
    
    public BookMyShow(int n, int m) {
        this.n = n;
        this.m = m;
        
        int size = 4 * n;
        maxFree = new long[size];
        totalFree = new long[size];
        
        Build(0, 0, n - 1, m);
    }
    
    private void Build(int node, int left, int right, long val) {
        if (left == right) {
            maxFree[node] = val;
            totalFree[node] = val;
            return;
        }
        
        int mid = (left + right) / 2;
        Build(2 * node + 1, left, mid, val);
        Build(2 * node + 2, mid + 1, right, val);
        
        maxFree[node] = Math.Max(maxFree[2 * node + 1], maxFree[2 * node + 2]);
        totalFree[node] = totalFree[2 * node + 1] + totalFree[2 * node + 2];
    }
    
    public int[] Gather(int k, int maxRow) {
        // Find first row with >= k consecutive seats
        int row = QueryMaxRow(0, 0, n - 1, 0, maxRow, k);
        
        if (row == -1) return new int[0];
        
        // Get current free seats in this row
        long free = QueryMaxSingle(0, 0, n - 1, row);
        int startSeat = (int)(m - free);
        
        // Update row
        Update(0, 0, n - 1, row, -k);
        
        return new int[] { row, startSeat };
    }
    
    public bool Scatter(int k, int maxRow) {
        // Check if enough total seats
        long total = QueryTotal(0, 0, n - 1, 0, maxRow);
        
        if (total < k) return false;
        
        // Allocate seats greedily from row 0
        long remaining = k;
        for (int row = 0; row <= maxRow && remaining > 0; row++) {
            long free = QueryMaxSingle(0, 0, n - 1, row);
            long take = Math.Min(remaining, free);
            
            if (take > 0) {
                Update(0, 0, n - 1, row, -take);
                remaining -= take;
            }
        }
        
        return true;
    }
    
    // Find first row in [queryLeft, queryRight] with maxFree >= k
    private int QueryMaxRow(int node, int left, int right, int queryLeft, int queryRight, long k) {
        if (queryLeft > right || queryRight < left || maxFree[node] < k) {
            return -1;
        }
        
        if (left == right) {
            return left;
        }
        
        int mid = (left + right) / 2;
        
        // Try left child first
        int res = QueryMaxRow(2 * node + 1, left, mid, queryLeft, queryRight, k);
        if (res != -1) return res;
        
        // Try right child
        return QueryMaxRow(2 * node + 2, mid + 1, right, queryLeft, queryRight, k);
    }
    
    // Query total free seats in range
    private long QueryTotal(int node, int left, int right, int queryLeft, int queryRight) {
        if (queryLeft > right || queryRight < left) return 0;
        
        if (queryLeft <= left && right <= queryRight) {
            return totalFree[node];
        }
        
        int mid = (left + right) / 2;
        return QueryTotal(2 * node + 1, left, mid, queryLeft, queryRight) +
               QueryTotal(2 * node + 2, mid + 1, right, queryLeft, queryRight);
    }
    
    // Query max free in single row
    private long QueryMaxSingle(int node, int left, int right, int row) {
        if (left == right) {
            return maxFree[node];
        }
        
        int mid = (left + right) / 2;
        if (row <= mid) {
            return QueryMaxSingle(2 * node + 1, left, mid, row);
        } else {
            return QueryMaxSingle(2 * node + 2, mid + 1, right, row);
        }
    }
    
    // Update row by delta
    private void Update(int node, int left, int right, int row, long delta) {
        if (left == right) {
            maxFree[node] += delta;
            totalFree[node] += delta;
            return;
        }
        
        int mid = (left + right) / 2;
        if (row <= mid) {
            Update(2 * node + 1, left, mid, row, delta);
        } else {
            Update(2 * node + 2, mid + 1, right, row, delta);
        }
        
        maxFree[node] = Math.Max(maxFree[2 * node + 1], maxFree[2 * node + 2]);
        totalFree[node] = totalFree[2 * node + 1] + totalFree[2 * node + 2];
    }
}
```

## Complexity

| **Operation** | **Time** | **Space** |
|---|---|---|
| **Constructor** | O(n) | O(n) |
| **Gather** | O(log n) | - |
| **Scatter** | O(n log n) worst case | - |

---

## Key Takeaway

**Segment tree for efficient range queries and updates**. Track max consecutive seats (for gather) and total seats (for scatter). O(log n) per gather, O(n log n) worst scatter.
