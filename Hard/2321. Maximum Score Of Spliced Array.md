# 2321. Maximum Score Of Spliced Array
## O(n) — Kadane's Algorithm on Difference Array

---

## Problem Statement

- Swap one subarray between `nums1` and `nums2`. Maximize `max(sum(nums1), sum(nums2))` after the swap.

---

## Core Idea — Maximize Gain with Kadane's

Swapping subarray `[left..right]` from `nums2` into `nums1` increases `sum(nums1)` by `Σ(nums2[i]-nums1[i])` for `i ∈ [left..right]`.

This is exactly **maximum subarray sum of `nums2-nums1`** — solved by Kadane's algorithm!

So:
- `score1 = sum(nums1) + max_gain_from_replacing_with_nums2` = `sum1 + Kadane(nums2-nums1)`
- `score2 = sum(nums2) + max_gain_from_replacing_with_nums1` = `sum2 + Kadane(nums1-nums2)`

Answer = `max(score1, score2)`. The "no swap" case is handled by Kadane allowing gain of 0.

---

## Full Optimal Implementation (C++)

```cpp
class Solution {
public:
    int maximumsSplicedArray(vector<int>& nums1, vector<int>& nums2) {
        int sum1 = 0, sum2 = 0;
        for (int i = 0; i < (int)nums1.size(); i++) {
            sum1 += nums1[i];
            sum2 += nums2[i];
        }

        auto kadane = [&](vector<int>& a, vector<int>& b) {
            int cur = 0, best = 0;
            for (int i = 0; i < (int)a.size(); i++) {
                cur = max(0, cur + b[i] - a[i]);
                best = max(best, cur);
            }
            return best;
        };

        return max(sum1 + kadane(nums1, nums2),
                   sum2 + kadane(nums2, nums1));
    }
};
```

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n)** — two Kadane passes |
| **Space** | **O(1)** |

---

## Example Walkthrough

**`nums1=[60,60,60], nums2=[10,90,10]`**

sum1=180, sum2=110.

Kadane(nums1←nums2): diffs=[10-60,90-60,10-60]=[-50,30,-50]. Max subarray=30.
score1 = 180+30 = **210** ✓.

Kadane(nums2←nums1): diffs=[60-10,60-90,60-10]=[50,-30,50]. Max subarray=70 (not contiguous since -30 drags it). Actually max is max(50,50,70)=70.
score2 = 110+70 = 180.

Answer: **210** ✓.

---

## Pitfalls & Edge Cases

- **`cur = max(0, cur + diff)`**: allows restarting when gain is negative (no swap = gain 0).
- **Symmetric calls**: try replacing subarray in nums1 AND in nums2 — both could yield the maximum.
- **No swap**: when all differences are negative, Kadane returns 0 → score = original sum.

---

## Key Takeaway

"Max score after optional subarray swap" → **Kadane's on difference arrays. Gain from swap = max subarray of (nums2-nums1) or (nums1-nums2)**. Run twice, take max. O(n).
