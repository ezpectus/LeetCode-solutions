# LeetCode 3435. Frequencies of Shortest Supersequences

## 1. Constraints & Complexity Bottleneck Analysis

**Number of Strings (words.length):** `1 <= words.length <= 256`.

**String Length:** Strictly equal to 2 (`words[i].length == 2`). This means each word acts as a directed dependency edge `u -> v` between characters.

**Unique Characters Count (U):** `<= 16`. This extremely tight constraint is the pivotal structural exploit, opening the door for exponential bitmask brute-force over character roles.

### The Combinatorial Bottleneck

A Shortest Common Supersequence (SCS) must satisfy all ordering constraints given by the pairs. In graph theory, if all character constraints form a Directed Acyclic Graph (DAG), the shortest sequence uses each unique character exactly once. However, conflicting pairs like `"ab"` and `"ba"` introduce cycles. To break a cycle, at least one character in it must be duplicated (assigned a frequency of 2 instead of 1).

Since `U <= 16`, the brute force optimization space is `2^16 = 65,536`, which represents all possible subsets of characters to duplicate. The main bottleneck lies in validating whether a chosen subset of duplicated characters successfully breaks all cycles in `O(U + E)` time per state.

## 2. Algorithm Design & Mathematical Invariants

### The Graph Reduction & Node Splitting Invariant

Let `G = (V, E)` be a directed graph where `V` represents the unique characters present across all strings, and a directed edge `u -> v` exists if the string `u + v` is in `words`.

If we duplicate a node `u` (setting its frequency to 2), it means `u` can appear both at the beginning and at the end of a sequence, effectively absorbing incoming and outgoing edges without closing an internal cycle. Mathematically, duplicating a node removes it from cycle-dependency considerations. Thus, a choice of duplicated nodes is valid **if and only if** the subgraph induced by the remaining non-duplicated nodes (nodes with frequency 1) is completely acyclic (a DAG).

### Bitmask Optimization Pipeline

Instead of expensive dynamic programming on graphs, we leverage the tiny coordinate space using a bitmask of size `2^U`:

1. Identify all `U` unique characters and map them to indices `0 ... U-1`.
2. Filter the edges: If a word consists of identical characters (e.g., `"aa"`), that character must have a frequency of at least 2 if it interacts with other characters, or it inherently defaults to its native constraint. For distinct pairs `u -> v`, they form active edges.
3. Iterate through all masks from `0` to `2^U - 1`. A set bit `1` at position `i` means character `i` gets a frequency of 2. A clear bit `0` means it keeps a frequency of 1.
4. For each mask, run a Kahn's algorithm or DFS-based Cycle Detection over the remaining `0-bit` nodes. If a cycle is detected, reject the mask.
5. Track the minimum sequence length found. The length for any valid mask is:

```
Length = sum(frequencies) = U + popcount(mask)
```

6. Collect all frequency distributions matching the absolute minimum length, filter out duplicate frequency vectors, and return the result.

## 3. High-Performance C++ Implementation

```cpp
#include <vector>
#include <string>
#include <queue>
#include <algorithm>
#include <set>

using namespace std;

class Solution {
public:
    vector<vector<int>> supersequences(vector<string>& words) {
        // Step 1: Map unique characters to dense indices [0, U-1]
        vector<bool> present(26, false);
        for (const string& w : words) {
            present[w[0] - 'a'] = true;
            present[w[1] - 'a'] = true;
        }

        vector<int> char_to_idx(26, -1);
        vector<int> idx_to_char;
        int U = 0;
        for (int i = 0; i < 26; ++i) {
            if (present[i]) {
                char_to_idx[i] = U++;
                idx_to_char.push_back(i);
            }
        }

        // Step 2: Build the adjacency matrix for distinct character transitions
        vector<vector<bool>> adj(U, vector<bool>(U, false));
        vector<int> force_double(U, 0);

        for (const string& w : words) {
            int u = char_to_idx[w[0] - 'a'];
            int v = char_to_idx[w[1] - 'a'];
            if (u == v) {
                force_double[u] = 1;
            } else {
                adj[u][v] = true;
            }
        }

        int min_len = 1e9;
        vector<int> valid_masks;

        // Step 3: High-speed bitmask space exploration (2^U iterations)
        int max_mask = 1 << U;
        for (int mask = 0; mask < max_mask; ++mask) {
            // Prune masks that do not satisfy self-loop constraints
            bool feasible = true;
            for (int i = 0; i < U; ++i) {
                if (force_double[i] && !(mask & (1 << i))) {
                    feasible = false;
                    break;
                }
            }
            if (!feasible) continue;

            // Compute current total string length
            int current_len = U + __builtin_popcount(mask);
            if (current_len > min_len) continue; // Early pruning optimization

            // Validate if the remaining 1-frequency nodes form a DAG
            if (isAcyclic(mask, adj, U)) {
                if (current_len < min_len) {
                    min_len = current_len;
                    valid_masks.clear();
                }
                valid_masks.push_back(mask);
            }
        }

        // Step 4: Map optimal masks back to the 26-element frequency structure
        set<vector<int>> unique_freqs;
        for (int mask : valid_masks) {
            vector<int> freq(26, 0);
            for (int i = 0; i < U; ++i) {
                freq[idx_to_char[i]] = (mask & (1 << i)) ? 2 : 1;
            }
            unique_freqs.insert(freq);
        }

        return vector<vector<int>>(unique_freqs.begin(), unique_freqs.end());
    }

private:
    // Topologically check if the induced subgraph of 0-bits is a DAG
    bool isAcyclic(int mask, const vector<vector<bool>>& adj, int U) {
        vector<int> in_degree(U, 0);
        
        // Compute in-degrees considering only nodes that aren't duplicated (0 in mask)
        for (int u = 0; u < U; ++u) {
            if (mask & (1 << u)) continue;
            for (int v = 0; v < U; ++v) {
                if (mask & (1 << v)) continue;
                if (adj[u][v]) {
                    in_degree[v]++;
                }
            }
        }

        // Classical Kahn's Processing Queue
        queue<int> q;
        int processed_nodes = 0;
        int active_nodes = 0;

        for (int i = 0; i < U; ++i) {
            if (!(mask & (1 << i))) {
                active_nodes++;
                if (in_degree[i] == 0) {
                    q.push(i);
                }
            }
        }

        while (!q.empty()) {
            int u = q.front();
            q.pop();
            processed_nodes++;

            for (int v = 0; v < U; ++v) {
                if (mask & (1 << v)) continue;
                if (adj[u][v]) {
                    if (--in_degree[v] == 0) {
                        q.push(v);
                    }
                }
            }
        }

        return processed_nodes == active_nodes;
    }
};
```

## 4. Comprehensive Complexity & Mathematical Analysis

### Detailed Time Complexity Decomposition

The computational workflow consists of graph construction followed by a combinatorial state space search:

```
T_total = T_graph + T_search + T_export
```

**Graph Construction (T_graph):**

Scanning the input strings list takes linear time proportional to the number of words: `O(W)`, where `W = words.length <= 256`.

**State Space Cycle Verification (T_search):**

The search loops through `2^U` states. Inside each state, checking if the graph is acyclic via Kahn's algorithm iterates over all active nodes and edges in the induced subgraph. The maximum number of vertices is `U` and edges is bounded by `U^2`.

```
T_search = O(2^U * U^2)
```

Given `U <= 16`, `2^16 * 256 = 65,536 * 256 ≈ 1.67 × 10^7` operations, which executes well within the 1-second runtime budget.

**Deduplication and Export (T_export):**

Converting optimal masks to vectors and sorting them inside a standard balanced tree `std::set` takes `O(M * 26 * log M)` time, where `M` is the number of optimal solutions found (`M << 2^U`).

Combining all phases, the total time complexity is bounded by:

```
T_total = O(W + 2^U * U^2)
```

### Detailed Space Complexity Decomposition

The space footprint is dominated by the dense graph representation and bitmask filtering storage:

- **Adjacency Matrix Mapping:** An explicit graph representation requires `O(U^2)` space.
- **In-Degree and Mapping Storage:** Auxiliary arrays `char_to_idx`, `idx_to_char`, and `in_degree` scale linearly with the alphabet size: `O(U + 26)`.
- **BFS Processing Queue:** The Kahn's algorithm execution queue stores at most `U` elements at any given execution cycle.
- **Unique Frequency Output Storage:** The `std::set` tracks unique answer vectors, bounded by `O(M * 26)` space.

Total auxiliary memory limits are safely confined within a small linear footprint:

```
S_total = O(U^2 + 26)
```

## 5. Mathematical Formulations & Foundational Algorithms

### The Topological Invariant of Shortest Common Supersequences

A sequence `S` is a common supersequence of a set of directed constraint edges `E` if the order of elements in `S` does not violate the direction of any edge `(u, v) ∈ E`.

For a subset of unique characters `V`, if the graph `G = (V, E)` contains no cycles, there exists at least one permutation of `V` that satisfies all edges (obtained via topological sorting). The length of this sequence is exactly `|V|`.

If a set of cycles `C_1, C_2, ..., C_k` exists, no single permutation can satisfy the constraints. By assigning a character `u ∈ C_i` a frequency of 2, we split its instances into `u_first` and `u_last`, transforming the cycle into a linear path:

```
u_first -> ... -> v -> u_last
```

Thus, minimizing the sequence length is exactly equivalent to finding the minimum size **Feedback Vertex Set (FVS)** on the character constraint graph, which we solve optimally using explicit bitmask evaluation.
