# 2167. Minimum Time to Remove All Cars Containing Illegal Goods
## O(n) — Left/Right DP + Split Point Minimization

---

## Problem Statement

- Remove all '1' cars. Cost: 1 from left end, 1 from right end, 2 from middle. Minimize total time.

---

## Core Idea — Split into Left/Right Prefix + Suffix DP

In any optimal solution, some prefix of '1's is removed from the left, some suffix from the right, and the rest from the middle (cost 2 each).

**`left[i]`** = min cost to remove all '1's in `s[0..i]`:
- If `s[i]='0'`: `left[i] = left[i-1]`
- If `s[i]='1'`: `left[i] = min(i+1, left[i-1]+2)` — either remove everything up to i from the left (cost i+1), or remove via middle (cost +2 on top of previous).

**`right[i]`** = min cost to remove all '1's in `s[i..n-1]` (symmetric).

**Answer**: `min over all split points: left[i] + right[i+1]`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinimumTime(string s) {
        int n = s.Length;
        int[] left = new int[n], right = new int[n];

        left[0] = s[0]=='1' ? 1 : 0;
        for (int i = 1; i < n; i++)
            left[i] = s[i]=='0' ? left[i-1] : Math.Min(i+1, left[i-1]+2);

        right[n-1] = s[n-1]=='1' ? 1 : 0;
        for (int i = n-2; i >= 0; i--)
            right[i] = s[i]=='0' ? right[i+1] : Math.Min(n-i, right[i+1]+2);

        int res = Math.Min(left[n-1], right[0]);
        for (int i = 0; i < n-1; i++)
            res = Math.Min(res, left[i] + right[i+1]);
        return res;
    }
}
```

## Example Walkthrough

**`s="1100101"`**

left: [1,2,2,2,2,3,3] (costs to clear prefix).
right: [5,4,3,3,2,2,1] (costs to clear suffix).

Split at i=1: left[1]+right[2] = 2+3 = **5** ✓

---

## Pitfalls & Edge Cases

- **`min(i+1, left[i-1]+2)`**: `i+1` = take everything from the left end. This might be cheaper than middle-removing if '1's are clustered at the start.
- **Split at -1 (no left)**: use `right[0]`. **Split at n (no right)**: use `left[n-1]`. Both handled in initialization of `res`.

---

## Key Takeaway

"Min cost to clear all '1's with 3 operation types" → **prefix/suffix DP + split minimization**:

```
left[i] = min(i+1, left[i-1]+2) when s[i]='1'
answer = min over i: left[i] + right[i+1]
```

The split point separates "removed from left" and "removed from right/middle" regions.
