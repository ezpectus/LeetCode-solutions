# 1761. Minimum Degree of a Connected Trio in a Graph
## O(n³) — Adjacency Matrix + Triple Enumeration

---

## Problem Statement

- **Connected trio**: 3 nodes where every pair is connected.
- **Degree of a trio**: edges with exactly one endpoint inside the trio.
- Return minimum trio degree, or -1 if no trio exists.

---

## Core Idea — Enumerate All Triples + Degree Formula

**Enumerate** all triples `(a, b, c)` with `a < b < c`. Check if all three edges exist using an adjacency matrix (O(1) lookup). If yes, compute the degree.

**Degree formula**: each node in the trio has `deg[v]` total edges. The 3 internal edges (a-b, b-c, a-c) are counted twice (once per endpoint) = 6 total. Subtract them:

```
degree(a,b,c) = deg[a] + deg[b] + deg[c] - 6
```

**Why -6?** Each of the 3 internal edges appears in two nodes' degree counts → 3×2 = 6 to subtract.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinTrioDegree(int n, int[][] edges) {
        bool[,] adj = new bool[n+1, n+1];
        int[] deg = new int[n+1];

        foreach (var e in edges) {
            adj[e[0], e[1]] = adj[e[1], e[0]] = true;
            deg[e[0]]++; deg[e[1]]++;
        }

        int res = int.MaxValue;
        for (int a = 1; a <= n; a++)
            for (int b = a+1; b <= n; b++) {
                if (!adj[a, b]) continue;
                for (int c = b+1; c <= n; c++)
                    if (adj[a, c] && adj[b, c])
                        res = Math.Min(res, deg[a]+deg[b]+deg[c]-6);
            }

        return res == int.MaxValue ? -1 : res;
    }
}
```

## Complexity

| **Metric** | **Value** | **Notes** |
|---|---|---|
| **Time** | **O(n³)** | Triple nested loop; n≤400 → 64M ops |
| **Space** | **O(n²)** | Adjacency matrix |

`n=400` → `400³/6 ≈ 10⁷` unique triples — fast enough.

---

## Why Degree Formula Works

```
deg[a] = (edges to b) + (edges to c) + (edges to outside)
deg[b] = (edges to a) + (edges to c) + (edges to outside)
deg[c] = (edges to a) + (edges to b) + (edges to outside)

Sum = 2*(a-b) + 2*(b-c) + 2*(a-c) + outside_edges
    = 6 + outside_edges

∴ outside_edges = deg[a]+deg[b]+deg[c] - 6
```

---

## Pitfalls & Edge Cases

- **`if (!adj[a,b]) continue`** — skip inner loop entirely when `a-b` edge missing; saves ~half iterations
- **`-1` if no trio** — `res` stays `MaxValue` → return -1
- **1-indexed nodes** — allocate arrays of size `n+1`

---

## Key Takeaway

"Minimum degree connected trio" → **O(n³) enumeration with adjacency matrix**:

```
for a < b < c:
  if adj[a][b] && adj[a][c] && adj[b][c]:
    degree = deg[a]+deg[b]+deg[c]-6
```

The `-6` formula elegantly removes internal edges from degree counts. O(n³) is acceptable here since n≤400.
