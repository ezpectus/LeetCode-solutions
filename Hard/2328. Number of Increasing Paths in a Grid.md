# 2328. Number of Increasing Paths in a Grid

## O(m × n) — DFS with Memoization

---

## Problem Statement

- Given m×n integer grid.
- Count strictly **increasing paths** (can start/end anywhere, move 4-directionally).
- Return count modulo 10⁹+7.
- **Constraints**: 1 ≤ m, n ≤ 1000, 1 ≤ m×n ≤ 10⁵, 1 ≤ grid[i][j] ≤ 10⁵

---

## Core Idea — DFS with Memoization

**Key observation**: For each cell, count how many increasing paths **start** from that cell.

**DP definition**: `dp[r][c]` = number of increasing paths starting from (r,c).

**Base case**: Every cell is a path of length 1 → `dp[r][c] ≥ 1`.

**Recurrence**: From (r,c), can move to neighbor (nr,nc) if `grid[nr][nc] > grid[r][c]`:
```
dp[r][c] = 1 + sum(dp[nr][nc]) for all valid neighbors
```

**Answer**: Sum of `dp[r][c]` for all cells.

---

## Algorithm

```
memo = {}

DFS(r, c):
  if (r,c) in memo: return memo[(r,c)]
  
  count = 1 // Cell itself is a path
  
  for each neighbor (nr, nc) in 4 directions:
    if valid and grid[nr][nc] > grid[r][c]:
      count += DFS(nr, nc)
  
  memo[(r,c)] = count
  return count

Answer = sum(DFS(r, c) for all (r,c))
```

---

## Full Implementation (C#)

```csharp
public class Solution {
    private const int MOD = 1_000_000_007;
    private int[][] grid;
    private int m, n;
    private int?[,] memo;
    private int[][] dirs = new int[][] {
        new int[] {-1, 0}, new int[] {1, 0}, 
        new int[] {0, -1}, new int[] {0, 1}
    };
    
    public int CountPaths(int[][] grid) {
        this.grid = grid;
        m = grid.Length;
        n = grid[0].Length;
        memo = new int?[m, n];
        
        long totalPaths = 0;
        
        for (int r = 0; r < m; r++) {
            for (int c = 0; c < n; c++) {
                totalPaths = (totalPaths + DFS(r, c)) % MOD;
            }
        }
        
        return (int)totalPaths;
    }
    
    private int DFS(int r, int c) {
        if (memo[r, c].HasValue) {
            return memo[r, c].Value;
        }
        
        long count = 1; // Cell itself
        
        foreach (var dir in dirs) {
            int nr = r + dir[0];
            int nc = c + dir[1];
            
            if (nr >= 0 && nr < m && nc >= 0 && nc < n && 
                grid[nr][nc] > grid[r][c]) {
                count = (count + DFS(nr, nc)) % MOD;
            }
        }
        
        memo[r, c] = (int)count;
        return (int)count;
    }
}
```

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(m × n)** |
| **Space** | **O(m × n)** |

Each cell computed once due to memoization.

## Example Walkthrough

### Example 1: `grid = [[1,1],[3,4]]`

**DFS(0,0)** (value=1):
- Can move to (1,0) where value=3 > 1
- DFS(1,0) = ?

**DFS(1,0)** (value=3):
- Can move to (1,1) where value=4 > 3
- DFS(1,1) = 1 (no valid moves from 4)
- DFS(1,0) = 1 + 1 = 2

**Back to DFS(0,0)**:
- DFS(0,0) = 1 + 2 = 3

**DFS(0,1)** (value=1):
- Can move to (1,1) where value=4 > 1
- DFS(1,1) = 1 (already computed)
- DFS(0,1) = 1 + 1 = 2

**DFS(1,0)** = 2 (already computed)

**DFS(1,1)** = 1 (already computed)

**Total**: 3 + 2 + 2 + 1 = 8 ✓

---

## Key Takeaway

**DFS with memoization on DAG** (grid values define topological order). Count paths starting from each cell, sum all results. O(m×n) time and space.
