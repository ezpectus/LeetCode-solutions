# 3548. Equal Sum Grid Partition II
## O(m×n) — Prefix Sums + Connectivity Analysis + Value Range Lookups

---

## Problem Statement

- One horizontal or vertical cut. Both sections must be non-empty.
- Sections can be equalized by **discounting at most one cell** from either section.
- Discounted cell must not disconnect its section.
- Return true if such a cut exists.

---

## Core Idea — Three Phases

### Phase 1: Compute Prefix Sums

For horizontal cut after row `i`:
- `left_sum = sum(rows 0..i)`, `right_sum = total - left_sum`
- `diff = left_sum - right_sum`

If `diff == 0` → valid immediately. Otherwise need to remove a cell with value `|diff|` from the **larger** section.

### Phase 2: Connectivity Analysis

When can we remove a cell from a rectangular section without disconnecting it?

A section is a full rectangle (all cells present). Key insight:
- **≥ 2×2 rectangle**: removing ANY single cell keeps it connected (can always route around)
- **1×k strip**: removing an interior cell (not at either end) disconnects it → only endpoints `j=0` or `j=k-1` are safe
- **k×1 strip**: similarly, only endpoints `r=0` or `r=k-1` are safe

For horizontal cut after row `i`, left section has shape `(i+1) × n`:
- If `(i+1) ≥ 2` and `n ≥ 2`: any cell in the section works
- If `i+1 == 1` (single row): only cells at `col=0` or `col=n-1`
- If `n == 1` (single column): only cells at `row=0` or `row=i`

### Phase 3: Value Existence Check

For each cut, we need to know if a value `target = |diff|` exists in the valid region. Precompute for each value `v`:
- `val_min_row[v]`, `val_max_row[v]`: min/max row containing `v`
- `val_min_col[v]`, `val_max_col[v]`: min/max column containing `v`
- `val_min/max_corner_row[v]`: min/max row where `v` appears at `col=0` or `col=n-1` (for the 1-row edge case)
- `val_min/max_corner_col[v]`: similar for columns (for the 1-column edge case)

Existence check in row range `[lo,hi]`: `val_min_row[v] <= hi && val_max_row[v] >= lo` → O(1).

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public bool CanPartitionGrid(int[][] grid) {
        int m = grid.Length, n = grid[0].Length;
        long[] rowSum = new long[m], colSum = new long[n];
        var minRow = new Dictionary<int,int>(); var maxRow = new Dictionary<int,int>();
        var minCol = new Dictionary<int,int>(); var maxCol = new Dictionary<int,int>();
        var minCR = new Dictionary<int,int>(); var maxCR = new Dictionary<int,int>(); // corner rows
        var minCC = new Dictionary<int,int>(); var maxCC = new Dictionary<int,int>(); // corner cols

        long total = 0;
        for (int r = 0; r < m; r++)
            for (int c = 0; c < n; c++) {
                int v = grid[r][c];
                rowSum[r] += v; colSum[c] += v; total += v;
                if (!minRow.ContainsKey(v)) { minRow[v]=r; maxRow[v]=r; }
                else { minRow[v]=Math.Min(minRow[v],r); maxRow[v]=Math.Max(maxRow[v],r); }
                if (!minCol.ContainsKey(v)) { minCol[v]=c; maxCol[v]=c; }
                else { minCol[v]=Math.Min(minCol[v],c); maxCol[v]=Math.Max(maxCol[v],c); }
                if (c==0||c==n-1) {
                    if (!minCR.ContainsKey(v)) { minCR[v]=r; maxCR[v]=r; }
                    else { minCR[v]=Math.Min(minCR[v],r); maxCR[v]=Math.Max(maxCR[v],r); }
                }
                if (r==0||r==m-1) {
                    if (!minCC.ContainsKey(v)) { minCC[v]=c; maxCC[v]=c; }
                    else { minCC[v]=Math.Min(minCC[v],c); maxCC[v]=Math.Max(maxCC[v],c); }
                }
            }

        bool InRows(int v, int lo, int hi) =>
            minRow.ContainsKey(v) && minRow[v]<=hi && maxRow[v]>=lo;
        bool InCols(int v, int lo, int hi) =>
            minCol.ContainsKey(v) && minCol[v]<=hi && maxCol[v]>=lo;
        bool CornerRow(int v, int lo, int hi) =>
            minCR.ContainsKey(v) && minCR[v]<=hi && maxCR[v]>=lo;
        bool CornerCol(int v, int lo, int hi) =>
            minCC.ContainsKey(v) && minCC[v]<=hi && maxCC[v]>=lo;

        long pref = 0;
        for (int i = 0; i < m-1; i++) {
            pref += rowSum[i];
            long diff = pref - (total - pref);
            if (diff == 0) return true;
            long absDiff = Math.Abs(diff);
            if (absDiff > 100000) continue; // no single cell covers this → skip
            int t = (int)absDiff;
            if (diff > 0) { // remove from left (rows 0..i)
                int lr = i+1;
                if (lr>=2&&n>=2&&InRows(t,0,i)) return true;
                if (lr==1&&CornerRow(t,0,0)) return true;
                if (n==1&&(InRows(t,0,0)||InRows(t,i,i))) return true;
            } else { // remove from right (rows i+1..m-1)
                int rr = m-i-1;
                if (rr>=2&&n>=2&&InRows(t,i+1,m-1)) return true;
                if (rr==1&&CornerRow(t,m-1,m-1)) return true;
                if (n==1&&(InRows(t,i+1,i+1)||InRows(t,m-1,m-1))) return true;
            }
        }

        pref = 0;
        for (int j = 0; j < n-1; j++) {
            pref += colSum[j];
            long diff = pref - (total - pref);
            if (diff == 0) return true;
            long absDiff2 = Math.Abs(diff);
            if (absDiff2 > 100000) continue;
            int t = (int)absDiff2;
            if (diff > 0) {
                int lc = j+1;
                if (m>=2&&lc>=2&&InCols(t,0,j)) return true;
                if (lc==1&&CornerCol(t,0,0)) return true;
                if (m==1&&(InCols(t,0,0)||InCols(t,j,j))) return true;
            } else {
                int rc = n-j-1;
                if (m>=2&&rc>=2&&InCols(t,j+1,n-1)) return true;
                if (rc==1&&CornerCol(t,n-1,n-1)) return true;
                if (m==1&&(InCols(t,j+1,j+1)||InCols(t,n-1,n-1))) return true;
            }
        }

        return false;
    }
}
```

## Connectivity — Key Cases Illustrated

For a horizontal cut, left section = `(i+1) × n` rectangle:

```
Case: i+1 ≥ 2, n ≥ 2  (full 2D rectangle)
┌─────────────┐
│  . . . . .  │  ← Remove ANY cell, still connected
│  . . . . .  │    (routes around via other cells)
└─────────────┘

Case: i+1 = 1  (single row)
┌─────────────┐
│  X . . . X  │  ← Only corners (X) safe to remove
└─────────────┘    Interior removes split the strip

Case: n = 1  (single column)
┌─┐
│X│  ← Only top or bottom (X) safe
│ │
│ │
│X│
└─┘
```

## Example Walkthrough

**Example 3**: `grid=[[1,2,4],[2,3,5]]`

Horizontal cut after row 0: left=[1,2,4] sum=7, right=[2,3,5] sum=10, diff=-3.
Need to remove `3` from right section (rows 1..1 = single row, n=3 ≥ 2).
Single-row → only corners (col=0 or col=2). Grid[1] = [2,3,5]. `3` is at col=1 (interior) → NOT removable.
`5` at col=2 is a corner but `5 ≠ 3`.
→ Not valid.

No vertical cut works either → **false** ✓

## Pitfalls & Edge Cases

- **`diff` can be large** (`long` needed): total up to `10⁵ × 10⁵ = 10¹⁰`, diff up to that
- **`|diff|` might exceed `int` range**: guard with `Math.Abs(diff)` as `long`, cast to `int` only after verifying ≤ max value
- **`val_min_row[v] <= hi && val_max_row[v] >= lo`** checks if any occurrence falls in range — doesn't verify a specific occurrence exists in the range, but given min ≤ hi and max ≥ lo, at least one must
- **Both sections must be non-empty**: loop from `i=0` to `m-2` ensures both sections have ≥ 1 row

---

## Key Takeaway

Hard II version of Equal Sum Grid Partition with discounting — three ingredients:

1. **Prefix sums** → compute `diff` per cut in O(1)
2. **Connectivity analysis** → rectangle cases (2D vs 1D strip) determine which cells are removable
3. **Min/max row/col per value** → O(1) existence check in any row/column range

The connectivity insight (2D rectangle tolerates any removal; 1D strip only tolerates endpoints) is the crux of the hard difficulty.
