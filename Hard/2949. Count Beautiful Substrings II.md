# 2949. Count Beautiful Substrings II
## O(n + k) вҖ” Prefix Balance Transform + vВІвүЎ0(mod k) вҶ” p|v Reduction вҶ’ Modular Hash Map

---

## Problem Statement

String `s`, integer `k`. A substring is **beautiful** if:
1. `vowels == consonants`
2. `vowels Г— consonants вүЎ 0 (mod k)`

Return the count of beautiful substrings.

---

## Core Insight вҖ” Two Conditions, One Unified State

### Condition 1 вҶ’ Prefix Sum Balance

Transform each character: `+1` if vowel, `вҲ’1` if consonant. Let `prefix[i]` = transformed sum from index 0 to iвҲ’1 (`prefix[0] = 0`).

Substring `[l, r]` has `vowels = consonants` iff the sum of its transformed values is 0:
```
prefix[r+1] вҲ’ prefix[l] = 0   вҶ”   prefix[r+1] = prefix[l]
```

Let `v = vowels = consonants`. Substring length = `r вҲ’ l + 1 = 2v`.

### Condition 2 вҶ’ Period Constraint

Since `vowels = consonants = v`:
```
vowels Г— consonants = vВІ вүЎ 0 (mod k)
```

**Key reduction (number theory):** `vВІ вүЎ 0 (mod k)` if and only if `p | v`, where `p` is the smallest positive integer with `pВІ вүЎ 0 (mod k)`.

**Proof:** Factorize `k = вҲҸ pбөў^eбөў`. Then:
```
k | vВІ   вҶ”   pбөў^eбөў | vВІ  for all i
         вҶ”   pбөў^вҢҲeбөў/2вҢү | v  for all i     [since ordвӮӮ(vВІ) = 2В·ordвӮӮ(v)]
         вҶ”   p | v          where p = вҲҸ pбөў^вҢҲeбөў/2вҢү
```

So valid lengths are exactly multiples of `2p`. This converts a quadratic modular condition into a linear divisibility condition.

### Combined: The State Key

A substring `[l, r]` is beautiful iff:
1. `prefix[r+1] = prefix[l]`       (equal vowels/consonants)
2. `(r+1 вҲ’ l) вүЎ 0 (mod 2p)`       (valid length = multiple of 2p)

Equivalently: `(r+1)` and `l` share the same **state = (prefix value, index mod 2p)**.

Count pairs (l, r+1) with identical state using a hash map. One pass over the string.

---

## Computing p

`p` = smallest positive integer with `pВІ % k = 0`. Since `p вүӨ k вүӨ 1000`, a direct search is O(k):

```csharp
int p = 1;
while (p * p % k != 0) p++;   // terminates at p вүӨ k
```

**Examples:**
- k=1: p=1 (1ВІ%1=0)
- k=2: p=2 (1ВІ%2=1вү 0; 2ВІ%2=0)
- k=4: p=2 (4%4=0)
- k=12 = 2ВІГ—3: p = 2^вҢҲ2/2вҢү Г— 3^вҢҲ1/2вҢү = 2Г—3 = 6
- k=1000 = 2ВіГ—5Ві: p = 2^вҢҲ3/2вҢү Г— 5^вҢҲ3/2вҢү = 4Г—25 = 100

---

## Algorithm

```
1. Find p = min{v вүҘ 1 : vВІ % k = 0}.  Set period = 2p.
2. Initialize map: {(0, 0) вҶ’ 1}.  [prefix=0 before index 0]
3. For i = 0 to nвҲ’1:
     a. prefix += (IsVowel(s[i]) ? +1 : вҲ’1)
     b. key = (prefix, (i+1) % period)
     c. ans += map[key]       (pairs (l, i+1) sharing this state)
     d. map[key]++
4. Return ans.
```

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public long BeautifulSubstrings(string s, int k) {
        // Step 1: find smallest p with pВІ % k == 0
        int p = 1;
        while (p * p % k != 0) p++;
        int period = 2 * p;

        // Step 2: prefix balance + modular index hash map
        var dict = new Dictionary<(int prefix, int mod), long>();
        dict[(0, 0)] = 1;   // empty prefix at index 0

        long ans = 0;
        int prefix = 0;

        for (int i = 0; i < s.Length; i++) {
            prefix += IsVowel(s[i]) ? 1 : -1;
            var key = (prefix, (i + 1) % period);
            if (dict.TryGetValue(key, out long cnt)) ans += cnt;
            dict[key] = dict.GetValueOrDefault(key, 0L) + 1;
        }

        return ans;
    }

    static bool IsVowel(char c) =>
        c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
}
```

|---|---|---|

### Why the Period Matters: k=12 Illustration

k=12 = 2ВІГ—3. p = 2^вҢҲ2/2вҢү Г— 3^вҢҲ1/2вҢү = 2Г—3 = **6**. period = 12.

Valid vowel counts v:
- v=3: 9%12=9 вү  0 вң—
- v=4: 16%12=4 вү  0 вң—

Only substrings of length 12, 24, 36, ... (multiples of 2p=12) with equal vowels/consonants qualify. The period=12 key filters exactly these. вң“

## Pitfalls & Edge Cases

- **Finding p must not modify k.** The loop `while (p * p % k != 0) p++` uses `k` unchanged throughout. If you factorize by dividing k in-place, you lose the original k needed later. вң“

- **Loop terminates:** p=k always satisfies kВІ%k=0 (k divides kВІ). So p вүӨ k вүӨ 1000. No infinite loop. вң“

- **pВІ%k=0 does NOT mean p|k.** For k=6: p=6. 6%6=0 вҶ’ p|k coincidentally. For k=4: p=2. 4%2=0 but p|k again. For k=8: 1ВІ=1, 2ВІ=4, 3ВІ=9, 4ВІ=16; 16%8=0 вҶ’ p=4. 4 does not divide 8... actually 4 does divide 8. Hmm. For k=12: p=6, and 6 does not divide 12... actually 6|12. Let me try k=9: 1ВІ=1, 2ВІ=4, 3ВІ=9; 9%9=0 вҶ’ p=3. 3|9 yes. Counterexample: k=18=2Г—3ВІ. p=2^вҢҲ1/2вҢүГ—3^вҢҲ2/2вҢү=2Г—3=6. 36%18=0 вң“. Does 6|18? Yes. Hmm. Actually p always divides k (since p's prime factors are a subset of k's, with smaller or equal exponents). So p|k always.

- **The key condition is p|v, not p|k.** We need v (number of vowels) to be a multiple of p, not that p divides k.

- **k=1:** p=1, period=2. Any substring with equal vowels/consonants is valid (vВІ%1=0 always). Period=2 is automatically satisfied when prefix[r+1]=prefix[l] (both indexes have the same parity). So the period condition is redundant for k=1 but doesn't hurt. вң“

- **Prefix can be negative.** Use tuple key `(prefix, mod)` rather than treating prefix as an array index. A plain array of size n would need offset n to handle negatives; a dictionary is simpler. вң“

- **Overflow:** at most n(n+1)/2 pairs for k=1 with alternating vowels/consonants. For n=5Г—10вҒҙ: вүҲ 1.25Г—10вҒ№. Fits in `long`. Return type and `ans` must be `long`. вң“

## Key Takeaway

**Two-condition substring problem вҶ’ reduce each to a prefix property вҶ’ combine into one hash-map key.**

The two breakthrough steps:
1. **Vowels = consonants** вҶ’ prefix transform {vowel: +1, consonant: вҲ’1} вҶ’ look for pairs with equal prefix values.
2. **vВІ%k=0** вҶ’ number theory: v must be a multiple of the "root" p (smallest positive integer with pВІвүЎ0 mod k) вҶ’ the index mod 2p must match.

State key = **(prefix_value, index mod 2p)**. Each unique state seen `c` times contributes `cГ—(cвҲ’1)/2` pairs, counted incrementally.

**Pattern recognition:** "Count substrings satisfying condition A AND condition B" where A encodes as prefix equality and B encodes as index periodicity вҶ’ single-pass hash map on (prefix, index_mod_period).
