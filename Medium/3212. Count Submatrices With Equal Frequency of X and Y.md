# 3212. Count Submatrices With Equal Frequency of X and Y
## O(m × n) — Dual 2D Prefix Sum

---

## Problem Statement

- Count submatrices containing `(0,0)` where `count('X') == count('Y') > 0`.
- Same as 3070 but with two conditions tracked simultaneously.

---

## Core Idea — Two Parallel 2D Prefix Sums

Maintain two prefix sum tables simultaneously — `px[r][c]` for X counts and `py[r][c]` for Y counts — both representing the rectangle `[0,0]..[r,c]`.

Each cell `(r,c)` defines one submatrix containing `(0,0)`. Count it if:
- `px[r][c] == py[r][c]` (equal frequency)
- `px[r][c] > 0` (at least one X)

Build both prefix sums in a single pass using the standard 2D inclusion-exclusion formula.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int NumberOfSubmatrices(char[][] grid) {
        int m = grid.Length, n = grid[0].Length, res = 0;
        int[,] px = new int[m, n];
        int[,] py = new int[m, n];

        for (int r = 0; r < m; r++)
            for (int c = 0; c < n; c++) {
                px[r, c] = grid[r][c] == 'X' ? 1 : 0;
                py[r, c] = grid[r][c] == 'Y' ? 1 : 0;
                if (r > 0) { px[r,c] += px[r-1,c]; py[r,c] += py[r-1,c]; }
                if (c > 0) { px[r,c] += px[r,c-1]; py[r,c] += py[r,c-1]; }
                if (r > 0 && c > 0) { px[r,c] -= px[r-1,c-1]; py[r,c] -= py[r-1,c-1]; }
                if (px[r,c] == py[r,c] && px[r,c] > 0) res++;
            }

        return res;
    }
}
```

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(m × n)** |
| **Space** | **O(m × n)** — two prefix arrays |

---

## Relation to 3070

This is the same pattern as [3070](./3070_count_submatrices.md) but instead of checking `sum ≤ k`, we check `prefX == prefY > 0`. Two prefix tables instead of one, same O(m×n) approach.

---

## Key Takeaway

"Count top-left submatrices with equal counts of two characters" → **two parallel 2D prefix sums**, check condition per cell. Each `(r,c)` = one submatrix starting at `(0,0)`.
