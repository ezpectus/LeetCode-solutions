# 3254. Find the Power of K-Size Subarrays I
## O(n) — Consecutive Run Length Array

---

## Problem Statement

- Power of subarray = its max element if all elements are consecutive and sorted ascending, else -1.
- Return power of every k-size subarray.

---

## Core Idea — Precompute Consecutive Run Lengths

`consec[i]` = length of the consecutive ascending run ending at index `i`.

A k-size subarray `nums[i..i+k-1]` is valid iff `consec[i+k-1] >= k` — meaning the last `k` elements form a consecutive ascending sequence.

If valid, power = `nums[i+k-1]` (the maximum = last element of sorted consecutive run).

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int[] ResultsArray(int[] nums, int k) {
        int n = nums.Length;
        int[] consec = new int[n];
        consec[0] = 1;
        for (int i = 1; i < n; i++)
            consec[i] = nums[i] == nums[i-1]+1 ? consec[i-1]+1 : 1;

        int[] res = new int[n-k+1];
        for (int i = 0; i < n-k+1; i++)
            res[i] = consec[i+k-1] >= k ? nums[i+k-1] : -1;
        return res;
    }
}
```

## Key Takeaway

Precompute run lengths → check if run at the right end of window is ≥ k → O(n).
