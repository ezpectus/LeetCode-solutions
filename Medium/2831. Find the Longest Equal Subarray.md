# 2831. Find the Longest Equal Subarray
## O(n) — Group Positions by Value + Sliding Window

---

## Problem Statement

- Delete at most `k` elements. Find the longest subarray where all remaining elements are equal.
- "Subarray" = contiguous in the **original** array (deletions create gaps, but the kept elements form a contiguous window).

---

## Core Idea — Per-Value Sliding Window on Positions

Group indices by value. For each value, its positions form a list `[p₀, p₁, ..., pₘ]`. A window `[l, r]` over these positions means we keep `r-l+1` copies of the value. The span in the original array = `positions[r] - positions[l] + 1`. Deletions needed = span − count = `(positions[r] - positions[l] + 1) - (r - l + 1)`.

Slide the window: if deletions > k, shrink from the left.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int LongestEqualSubarray(IList<int> nums, int k) {
        var pos = new Dictionary<int, List<int>>();
        for (int i = 0; i < nums.Count; i++) {
            if (!pos.ContainsKey(nums[i])) pos[nums[i]] = new List<int>();
            pos[nums[i]].Add(i);
        }

        int res = 0;
        foreach (var positions in pos.Values) {
            int l = 0;
            for (int r = 0; r < positions.Count; r++) {
                // deletions = span - count = (pos[r]-pos[l]+1) - (r-l+1)
                while (positions[r] - positions[l] - (r - l) > k)
                    l++;
                res = Math.Max(res, r - l + 1);
            }
        }
        return res;
    }
}
```

---

## Complexity

| **Metric** | **Value** | **Notes** |
|---|---|---|
| **Time** | **O(n)** | Total positions across all values = n; each processed once |
| **Space** | **O(n)** | Position map |

---

## Deletion Formula Simplified

```
deletions = (positions[r] - positions[l] + 1) - (r - l + 1)
          = positions[r] - positions[l] - r + l
          = (positions[r] - r) - (positions[l] - l)
```

So the condition `> k` becomes `positions[r] - r - (positions[l] - l) > k`.

---

## Example Walkthrough

**`nums=[1,3,2,3,1,3], k=3`**

- `pos[3] = [1, 3, 5]`

| r | l | pos[r]-pos[l]-(r-l) | deletions | count |
|---|---|---------------------|-----------|-------|
| 0 | 0 | 0 | 0≤3 | 1 |
| 1 | 0 | 3-1-1=1 | 1≤3 | 2 |
| 2 | 0 | 5-1-2=2 | 2≤3 | **3** |

Max count = **3** ✓

---

## Key Takeaway

"Longest equal subarray with k deletions" → **group positions by value, sliding window over each group**:

```
deletions = span - count = (pos[r]-pos[l]+1) - (r-l+1)
valid window: deletions ≤ k
```

Total complexity O(n) since sum of all group sizes = n.
