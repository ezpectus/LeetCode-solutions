# 3334. Find the Maximum Factor Score of Array
## O(n² log max) — Try Removing Each Element

---

## Problem Statement

- Factor score = GCD(array) × LCM(array). Remove at most one element to maximize score.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    long Gcd(long a, long b) => b == 0 ? a : Gcd(b, a%b);
    long Lcm(long a, long b) => a / Gcd(a,b) * b;

    long Score(int[] nums, int skip) {
        long g = 0, l = 1;
        for (int i = 0; i < nums.Length; i++) {
            if (i == skip) continue;
            g = Gcd(g, nums[i]);
            l = Lcm(l, nums[i]);
        }
        return g == 0 ? 0 : g * l;
    }

    public long MaxScore(int[] nums) {
        long res = Score(nums, -1); // no removal
        for (int i = 0; i < nums.Length; i++)
            res = Math.Max(res, Score(nums, i));
        return res;
    }
}
```

**Time**: O(n² log max) — n+1 passes each O(n log max). Fine for n≤100.

---

## Key Takeaway

n≤100 → brute force all n+1 options (no removal + remove each element). GCD(0,x)=x handles initial accumulation.
