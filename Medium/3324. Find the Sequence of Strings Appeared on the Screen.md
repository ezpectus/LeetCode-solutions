# 3324. Find the Sequence of Strings Appeared on the Screen
## O(n × 26) — Simulate Key Presses

---

## Problem Statement

- Key1: append 'a'. Key2: increment last char. Build `target` with minimum presses, return all intermediate strings.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public IList<string> StringSequence(string target) {
        var res = new List<string>();
        var sb = new StringBuilder();

        foreach (char c in target) {
            sb.Append('a');           // Key1: append 'a'
            res.Add(sb.ToString());
            for (char cur = 'a'; cur < c; cur++) {
                sb[sb.Length-1]++;    // Key2: increment last char
                res.Add(sb.ToString());
            }
        }
        return res;
    }
}
```

**Time**: O(n × 26) — each character needs at most 25 increments. **Space**: O(n × 26) for output.

---

## Key Takeaway

Per character: append 'a', increment up to target char. At most 26 steps per character.
