# 2679. Sum in a Matrix
## O(m × n log n) — Sort Rows + Column Max

---

## Problem Statement

- Each round: remove the largest element from each row, add the max of those removed to score.
- Return total score.

---

## Core Idea

Sort each row ascending. Each "round" corresponds to one column index (rightmost = first round). The score contribution of column `c` = `max over all rows of nums[r][c]`.

After sorting rows, iterate columns right-to-left (or just sum column maxes in any order — result is the same).

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MatrixSum(int[][] nums) {
        int m = nums.Length, n = nums[0].Length;
        foreach (var row in nums) Array.Sort(row);

        int res = 0;
        for (int c = 0; c < n; c++) {
            int colMax = 0;
            for (int r = 0; r < m; r++)
                colMax = Math.Max(colMax, nums[r][c]);
            res += colMax;
        }
        return res;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(m × n log n)** — sorting each row |
| **Space** | **O(1)** — in-place sort |

---

## Key Takeaway

Sort rows → rounds become columns → score = sum of column maxes. The simulation collapses into two nested loops.
