# 3144. Minimum Substring Partition of Equal Character Frequency
## O(nВІ) вЂ” DP with O(1) Balance Check

---

## Problem Statement

- Partition string `s` into minimum number of **balanced** substrings.
- Balanced = all characters in the substring occur the same number of times.

---

## Core Idea вЂ” DP with Inner Loop Balance Check

**State**: `dp[i]` = minimum partitions for `s[0..i-1]`.

**Transition**: for each end position `i`, try every start `j в€€ [1..i]`:
```
if s[j-1..i-1] is balanced:
    dp[i] = min(dp[i], dp[j-1] + 1)
```

**Efficient balance check**: maintain `distinct` (number of unique chars) and `maxFreq` (max frequency) as we scan. A substring is balanced iff `distinct Г— maxFreq == length`. This works because if all frequencies equal `maxFreq`, then `sum = distinct Г— maxFreq = length`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinimumSubstringsInPartition(string s) {
        int n = s.Length;
        int[] dp = new int[n+1];
        Array.Fill(dp, n+1);
        dp[0] = 0;

        for (int i = 1; i <= n; i++) {
            int[] freq = new int[26];
            int distinct = 0, maxFreq = 0;

            for (int j = i; j >= 1; j--) {
                int c = s[j-1] - 'a';
                if (freq[c] == 0) distinct++;
                freq[c]++;
                maxFreq = Math.Max(maxFreq, freq[c]);

                int len = i - j + 1;
                if (distinct * maxFreq == len && dp[j-1] < n+1)
                    dp[i] = Math.Min(dp[i], dp[j-1] + 1);
            }
        }

        return dp[n];
    }
}
```

## Balance Check: `distinct Г— maxFreq == len`

A substring is balanced if all characters appear equally often. If they do, each appears `maxFreq` times, and `distinct Г— maxFreq = total length`. This is both necessary and sufficient.

Example: "ccdd" в†’ distinct=2, maxFreq=2, len=4. 2Г—2=4 вњ“. Balanced.
Example: "ccd" в†’ distinct=2, maxFreq=2, len=3. 2Г—2=4в‰ 3. Not balanced.

## Pitfalls & Edge Cases

- **Single char**: always balanced (distinct=1, maxFreq=1, len=1, 1Г—1=1 вњ“).
- **Scan backward** (j from i to 1): builds freq incrementally, avoids rebuilding for each j.
- **`dp[j-1] < n+1`**: guard against extending unreachable prefix states.

---

## Key Takeaway

"Min partitions into balanced substrings" в†’ **DP + `distinct Г— maxFreq == len` balance check**:

```
dp[i] = min over valid j of dp[j-1] + 1
balanced: distinct Г— maxFreq == i - j + 1
```

O(nВІ) total with O(1) balance check via incrementally maintained `distinct` and `maxFreq`.
