# 3556. Sum of Largest Prime Substrings  
## O(n² · log M) — Enumerate All Substrings + Deterministic 64‑bit Miller–Rabin

---

## Problem Overview

Given a digit string `s` (length ≤ 10), consider **all substrings**.  
Each substring is converted to an integer (leading zeros ignored).  
Among all unique primes formed this way:

- return the **sum of the 3 largest primes**,
- if fewer than 3 exist → sum all available,
- if none exist → return `0`.

---

## Key Insight

Since:

- `|s| ≤ 10`
- number of substrings ≤ 55
- max integer value < 10¹⁰ (fits in uint64)

We can brute‑force all substrings and check primality.

### ⚠ Important

C# version failed because it used Miller–Rabin bases `{2,7,61}`,  
which are valid **only for 32‑bit integers**.

For **64‑bit correctness**, we must use bases:

```
{2, 325, 9375, 28178, 450775, 9780504, 1795265022}
```

This guarantees correctness for all 64‑bit integers.

---

## Full C++ Implementation

```cpp
// millerRabin 
class Solution {
public:
    long long sumOfLargestPrimes(string s) {
        int n = s.size();
        unordered_set<unsigned long long> primes;

        for (int i = 0; i < n; i++) {
            unsigned long long val = 0;
            for (int j = i; j < n; j++) {
                val = val * 10 + (s[j] - '0');
                if (isPrime(val))
                    primes.insert(val);
            }
        }

        if (primes.empty())
            return 0;

        vector<unsigned long long> arr(primes.begin(), primes.end());
        sort(arr.begin(), arr.end(), greater<>());

        long long sum = 0;
        for (int i = 0; i < min(3, (int)arr.size()); i++)
            sum += arr[i];

        return sum;
    }

private:
    // 64-bit deterministic Miller–Rabin
    bool isPrime(unsigned long long n) {
        if (n < 2) return false;

        static unsigned long long smallPrimes[] =
            {2ULL,3ULL,5ULL,7ULL,11ULL,13ULL,17ULL,19ULL,23ULL,29ULL};

        for (unsigned long long p : smallPrimes) {
            if (n == p) return true;
            if (n % p == 0) return n == p;
        }

        return millerRabin(n);
    }

    bool millerRabin(unsigned long long n) {
        unsigned long long d = n - 1, s = 0;
        while ((d & 1) == 0) {
            d >>= 1;
            s++;
        }

        static unsigned long long bases[] =
            {2ULL, 325ULL, 9375ULL, 28178ULL,
             450775ULL, 9780504ULL, 1795265022ULL};

        for (unsigned long long a : bases) {
            if (a % n == 0) continue;
            if (!check(n, a, d, s)) return false;
        }
        return true;
    }

    bool check(unsigned long long n, unsigned long long a,
               unsigned long long d, unsigned long long s) {

        unsigned long long x = powmod(a, d, n);
        if (x == 1 || x == n - 1) return true;

        for (unsigned long long r = 1; r < s; r++) {
            x = mulmod(x, x, n);
            if (x == n - 1) return true;
        }
        return false;
    }

    unsigned long long mulmod(unsigned long long a,
                               unsigned long long b,
                               unsigned long long mod) {
        __uint128_t res = ( __uint128_t )a * b;
        return (unsigned long long)(res % mod);
    }

    unsigned long long powmod(unsigned long long a,
                               unsigned long long d,
                               unsigned long long mod) {
        unsigned long long r = 1;
        while (d > 0) {
            if (d & 1)
                r = mulmod(r, a, mod);
            a = mulmod(a, a, mod);
            d >>= 1;
        }
        return r;
    }
};
```

## Complexity

| Step | Cost |
|------|------|
| Enumerating substrings | O(n²) ≤ 55 |
| Primality test | O(log M) |
| Total | **O(n² log M)** — trivial for constraints |

## Example

`s = "6357501617"`

Largest prime substrings include:

```
6415753395
58753395
...
```

Top 3 sum = **6415753395** (correct).

---

## Key Takeaway

- The problem is small enough for brute force.
- The only tricky part is **correct 64‑bit primality testing**.
- Using the correct Miller–Rabin bases ensures full correctness.
