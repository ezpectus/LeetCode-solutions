# 2830. Maximize the Profit as the Salesman
## O(m log m + n) — DP with Sorted Offers

---

## Problem Statement

- `n` houses, numbered 0 to n-1. Each offer `[start, end, gold]` buys all houses in `[start,end]`.
- Non-overlapping offers only. Maximize total gold.

---

## Core Idea — Interval DP

**State**: `dp[i]` = maximum gold from houses `0..i-1`.

**Transition**:
- Don't sell house `i-1`: `dp[i] = dp[i-1]`
- Accept an offer `[s, e, g]` ending at `e = i-1`: `dp[i] = max(dp[i], dp[s] + g)`

Sort offers by end position. As we sweep `i` from 1 to n, process all offers ending at `i-1` and update `dp[i]`. The key: `dp[s]` is already computed (s < i), so we can use it directly.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MaximizeTheProfit(int n, IList<IList<int>> offers) {
        var sorted = offers.OrderBy(o => o[1]).ToList();
        int[] dp = new int[n + 1];
        int oi = 0;

        for (int i = 1; i <= n; i++) {
            dp[i] = dp[i-1]; // skip house i-1
            while (oi < sorted.Count && sorted[oi][1] == i-1) {
                int s = sorted[oi][0], g = sorted[oi][2];
                dp[i] = Math.Max(dp[i], dp[s] + g);
                oi++;
            }
        }

        return dp[n];
    }
}
```

## Complexity

| **Metric** | **Value** | **Notes** |
|---|---|---|
| **Time** | **O(m log m + n)** | Sort offers + linear sweep |
| **Space** | **O(n + m)** | dp array + sorted offers |

## Example Walkthrough

**`n=5, offers=[[0,0,1],[0,2,2],[1,3,2]]`** sorted by end: `[[0,0,1],[0,2,2],[1,3,2]]`

| i | dp[i-1] | offers ending at i-1 | dp[i] |
|---|---------|----------------------|-------|
| 1 | 0 | [0,0,1]: dp[0]+1=1 | **1** |
| 2 | 1 | none | 1 |
| 3 | 1 | [0,2,2]: dp[0]+2=2 | **2** |
| 4 | 2 | [1,3,2]: dp[1]+2=3 | **3** |
| 5 | 3 | none | 3 |

`dp[5] = 3` ✓

---

## Key Takeaway

"Max non-overlapping interval profit" → **sort by end, DP with `dp[start]` lookback**. Same pattern as weighted job scheduling. `dp[s] + g` means "best before this offer starts + this offer's value".
