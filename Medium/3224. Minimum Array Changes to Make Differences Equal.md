# 3224. Minimum Array Changes to Make Differences Equal
## O(n + k) — Difference Array on Savings

---

## Problem Statement

- Pair up `nums[i]` with `nums[n-1-i]`. Find X such that all pairs have `|a-b| = X`. Minimize total element changes (each change: replace any element with any value in `[0, k]`).

---

## Core Idea — Maximize Savings via Difference Array

For a fixed X, the cost per pair `(a, b)`:
- **0 changes** if `|a-b| == X`
- **1 change** if X is achievable by changing one element: `X ∈ [0, max(a, b, k-a, k-b)]`
- **2 changes** otherwise

**Strategy**: start with baseline cost = `n/2 × 2` (2 changes per pair). For each X, compute total "savings" = how much below the baseline we get. Find X with maximum savings.

**Savings per pair at X**:
- +1 for all `X ∈ [0, M]` where `M = max(a, b, k-a, k-b)` (saves 1 change)
- +1 extra at `X = |a-b|` (saves 1 more change, total 2 saved)

Use a **difference array** to apply range updates in O(1), then prefix sum to find the maximum.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinChanges(int[] nums, int k) {
        int n = nums.Length;
        int[] save = new int[k + 2];

        for (int i = 0; i < n / 2; i++) {
            int a = nums[i], b = nums[n-1-i];
            int d = Math.Abs(a - b);
            int M = Math.Max(Math.Max(a, b), Math.Max(k-a, k-b));

            // Range [0, M]: +1 saving (1 change instead of 2)
            save[0]++;
            if (M + 1 <= k) save[M+1]--;

            // At X = d: +1 extra saving (0 changes)
            save[d]++;
            if (d + 1 <= k) save[d+1]--;
        }

        int best = 0, cur = 0;
        for (int x = 0; x <= k; x++) {
            cur += save[x];
            best = Math.Max(best, cur);
        }

        return n / 2 * 2 - best;
    }
}
```

## Why M = max(a, b, k-a, k-b)?

To achieve `|a' - b'| = X` by changing only one element (say `a` → `a'`):
- `|a' - b| = X` → `a' = b ± X`. Must have `0 ≤ a' ≤ k`.
- Maximum achievable X with one change to `a`: max of `b` (set a'=0) and `k-b` (set a'=k). Similarly for changing `b`.
- So maximum X achievable with 1 change = `max(a, b, k-a, k-b)`.

---

## Example Walkthrough

**`nums=[1,0,1,2,4,3], k=4`**, n=6, pairs: (1,3),(0,4),(1,2).

Pair (1,3): d=2, M=max(1,3,3,1)=3. save[0..3]+=1, save[2]+=1(extra).
Pair (0,4): d=4, M=max(0,4,4,0)=4. save[0..4]+=1, save[4]+=1(extra).
Pair (1,2): d=1, M=max(1,2,3,2)=3. save[0..3]+=1, save[1]+=1(extra).

After prefix sum, X=2 gives max savings. Total min changes = 3×2 - max_savings = **2** ✓

---

## Key Takeaway

"Min changes for all pairs to have equal |difference|" → **difference array on savings, find X maximizing total savings**:

```
for each pair (a,b):
    range [0, M]: +1 saving
    point d=|a-b|: +1 extra saving
answer = n/2 * 2 - max_savings
```

O(n+k). The range+point update pattern via difference array is the key.
