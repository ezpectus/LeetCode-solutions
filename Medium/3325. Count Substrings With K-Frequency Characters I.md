# 3325. Count Substrings With K-Frequency Characters I
## O(n) — Sliding Window

---

## Problem Statement

- Count substrings where at least one character appears ≥ k times.

---

## Core Idea — Sliding Window + Count Extensions

For each `right`, when `freq[s[right]] >= k`, the window `[left, right]` is valid. All extensions `[left, right], [left, right+1], ..., [left, n-1]` are also valid (more chars can only help). Add `n - right` to result, then shrink from left.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int NumberOfSubstrings(string s, int k) {
        int n = s.Length, res = 0, left = 0;
        int[] freq = new int[26];

        for (int right = 0; right < n; right++) {
            freq[s[right]-'a']++;
            while (freq[s[right]-'a'] >= k) {
                res += n - right;
                freq[s[left]-'a']--;
                left++;
            }
        }
        return res;
    }
}
```

**Time**: O(n) — **Space**: O(1)

---

## Key Takeaway

Once window valid (`freq >= k`), add `n-right` and shrink left. Same pattern as 3297.
