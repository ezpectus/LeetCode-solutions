# 3030. Find the Grid of Region Average
## O(m×n×9) — Enumerate 3×3 Regions, Accumulate Averages

---

## Problem Statement

- A **valid region** is a 3×3 subgrid where every pair of adjacent pixels has absolute intensity difference ≤ `threshold`.
- Each pixel's result = floor average of all valid region averages it belongs to.
- If a pixel belongs to no region → keep original value.

---

## Core Idea — Enumerate All Valid 3×3 Subgrids

There are `(m-2) × (n-2)` possible 3×3 top-left corners. For each:
1. **Validate**: check all adjacent pairs (12 horizontal + 12 vertical = 24 pairs) within the subgrid.
2. **If valid**: compute the floor average (sum of 9 pixels / 9), add it to each of the 9 pixels' accumulator.

After processing all regions, for each pixel:
- If in ≥ 1 region: result = `floor(sum_of_region_averages / count_of_regions)`
- If in 0 regions: result = original `image[i][j]`

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int[][] ResultGrid(int[][] image, int threshold) {
        int m = image.Length, n = image[0].Length;
        int[,] rgnSum = new int[m, n];
        int[,] rgnCnt = new int[m, n];

        for (int r = 0; r <= m-3; r++) {
            for (int c = 0; c <= n-3; c++) {
                if (!IsValid(image, r, c, threshold)) continue;

                int total = 0;
                for (int i = 0; i < 3; i++)
                    for (int j = 0; j < 3; j++)
                        total += image[r+i][c+j];
                int avg = total / 9;

                for (int i = 0; i < 3; i++)
                    for (int j = 0; j < 3; j++) {
                        rgnSum[r+i, c+j] += avg;
                        rgnCnt[r+i, c+j]++;
                    }
            }
        }

        int[][] res = new int[m][];
        for (int i = 0; i < m; i++) {
            res[i] = new int[n];
            for (int j = 0; j < n; j++)
                res[i][j] = rgnCnt[i,j] == 0
                    ? image[i][j]
                    : rgnSum[i,j] / rgnCnt[i,j];
        }
        return res;
    }

    bool IsValid(int[][] img, int r, int c, int t) {
        for (int i = 0; i < 3; i++)
            for (int j = 0; j < 3; j++) {
                if (j+1 < 3 && Math.Abs(img[r+i][c+j] - img[r+i][c+j+1]) > t) return false;
                if (i+1 < 3 && Math.Abs(img[r+i][c+j] - img[r+i+1][c+j]) > t) return false;
            }
        return true;
    }
}
```

| **Metric** | **Value** | **Notes** |

| **Time** | **O(m×n×9)** = O(m×n) | (m-2)×(n-2) regions × 9 cells to check/update |
| **Space** | **O(m×n)** | Two accumulator arrays |

With `m,n ≤ 500`: at most ~500×500×9 ≈ 2.25M operations — instant.

---

## Two-Stage Rounding

The problem specifies a deliberate two-stage floor:

1. Each valid region's average is floored: `avg = sum / 9` (integer division).
2. A pixel's final value is the floor of the mean of floored averages: `result = sum_of_avgs / count` (integer division).

Both stages use floor division. This matches C#'s default integer division behavior.

## Example Walkthrough

**`image=[[5,6,7,10],[8,9,10,10],[11,12,13,10]], threshold=3`**

Valid 3×3 regions (top-left corners):
- **(0,0)**: check all adjacent pairs. Max diff = |7-10|=3 ≤ 3, |10-10|=0 ≤ 3, |13-10|=3 ≤ 3. ✓
  Sum = 5+6+7+8+9+10+11+12+13 = 81, avg = 9.
- **(0,1)**: includes 6,7,10,9,10,10,12,13,10. Check: |10-10|=0, |13-10|=3, |10-10|=0 ✓.
  Sum = 6+7+10+9+10+10+12+13+10 = 87, avg = 9.

All pixels belong to at least one region. Region averages: 9 and 9.
Result for all pixels = floor((9+9)/2) or floor(9/1) = **9** ✓.

## Pitfalls & Edge Cases

- **Adjacent pairs in 3×3**: 2 horizontal pairs per row × 3 rows = 6 horizontal, plus 2 vertical pairs per column × 3 columns = 6 vertical = 12 pairs total. The nested loop checking `j+1 < 3` and `i+1 < 3` covers all.
- **Pixel belongs to 0 regions**: copy original value. Track with `rgnCnt[i,j] == 0`.
- **Multiple regions overlap**: floor each region average first, then floor the mean of those floored averages. Don't use the raw sum.
- **`m=3, n=3`**: exactly one possible region (top-left at (0,0)). Works correctly.

---

## Key Takeaway

"Per-pixel average of valid 3×3 region averages" → **enumerate all (m-2)×(n-2) top-left corners, validate, accumulate floored averages**:

```
for each valid 3x3 region at (r,c):
    avg = sum_of_9_pixels / 9
    for each pixel in region: rgnSum[i][j] += avg; rgnCnt[i][j]++

result[i][j] = rgnCnt > 0 ? rgnSum / rgnCnt : image[i][j]
```

All divisions are integer (floor). O(m×n) total — straightforward implementation.
