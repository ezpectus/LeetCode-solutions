# 3690. Split and Merge Array Transformation

## O(n! × n³) — BFS State Space Search

---

## Problem Statement

- Given arrays `nums1` and `nums2` of length `n` (nums2 is permutation of nums1).
- **Operation**: Choose subarray [L..R], remove it, insert anywhere in remaining array.
- Return minimum operations to transform nums1 into nums2.
- **Constraints**: 2 ≤ n ≤ 6, -10⁵ ≤ nums[i] ≤ 10⁵

---

## What the Fuck is Split-Merge Operation?

**Example**: `nums = [3,1,2]`

**Operation breakdown**:
1. **Choose subarray**: Pick indices L=0, R=0 → subarray = [3]
2. **Remove it**: Array becomes [1,2] (removed [3])
3. **Insert anywhere**: Insert [3] at position 2 (after [1,2]) → [1,2,3]

**Another example**: `nums = [1,2,3,4,5]`
- Choose L=1, R=3 → subarray = [2,3,4]
- Remove: [1,5] remains
- Insert at position 0: [2,3,4,1,5]

**Key point**: You're cutting out a chunk and pasting it somewhere else, like cut-paste in text editor.

---

## Why BFS?

**This is a graph problem**:
- **Nodes** = all possible array configurations (permutations)
- **Edges** = one split-merge operation transforms one config to another
- **Goal** = shortest path from nums1 to nums2

**BFS guarantees**: First time we reach target = minimum operations (unweighted graph, all edges cost 1).

---

## How BFS Works Here

### State Representation

**State** = current array configuration  
**Example**: [3,1,2] is one state, [1,2,3] is another

We use **string representation** for easy comparison:
- [3,1,2] → "3,1,2"
- [1,2,3] → "1,2,3"

### BFS Structure

```
Queue: stores (current_array, number_of_operations)
Visited: set of already seen configurations (as strings)

Start:
  queue = [(nums1, 0)]
  visited = {nums1}

Loop until queue empty:
  Dequeue (current, ops)
  
  If current == nums2:
    Return ops  // Found shortest path!
  
  Generate all possible next states:
    For each possible subarray [L..R]:
      For each possible insert position:
        Create new state
        If new state not visited:
          Add to queue with (new_state, ops+1)
          Mark as visited
```

---

## Detailed Operation Generation

**For array `current` of length n**:

### Step 1: Choose Subarray

Try all possible subarrays [L..R]:
- L from 0 to n-1
- R from L to n-1

**Example**: n=3, possible subarrays:
- [0,0]: [current[0]]
- [0,1]: [current[0], current[1]]
- [0,2]: [current[0], current[1], current[2]]
- [1,1]: [current[1]]
- [1,2]: [current[1], current[2]]
- [2,2]: [current[2]]

### Step 2: Extract Subarray and Remaining

**Subarray** = current[L..R]  
**Remaining** = current[0..L-1] + current[R+1..n-1]

**Example**: current = [3,1,2,4], L=1, R=2
- Subarray = [1,2]
- Remaining = [3] + [4] = [3,4]

### Step 3: Try All Insert Positions

Insert subarray into `remaining` at all possible positions (0 to len(remaining)):

**Example**: remaining = [3,4], subarray = [1,2]
- Insert at 0: [1,2] + [3,4] = [1,2,3,4]
- Insert at 1: [3] + [1,2] + [4] = [3,1,2,4]
- Insert at 2: [3,4] + [1,2] = [3,4,1,2]

---

## Concrete Example Walkthrough

### Example: nums1 = [3,1,2], nums2 = [1,2,3]

**BFS Level 0** (ops = 0):
- Current state: [3,1,2]
- Target: [1,2,3]
- Not equal, continue

**Generate neighbors** (all possible operations):

1. **Subarray [3] (L=0,R=0)**:
   - Remaining: [1,2]
   - Insert at 0: [3,1,2] (same, skip)
   - Insert at 1: [1,3,2]
   - Insert at 2: [1,2,3] ✓ **FOUND TARGET!**

**Result**: 1 operation

---

## Complex Example: nums1 = [1,1,2,3,4,5], nums2 = [5,4,3,2,1,1]

**BFS Level 0**: [1,1,2,3,4,5]

**BFS Level 1**: Generate all 1-operation neighbors
- Try splitting [1], [1,1], [1,1,2], [2], [2,3], etc.
- Inserting at different positions
- Get states like: [2,1,1,3,4,5], [1,2,1,3,4,5], [1,3,4,5,1,2], etc.

**BFS Level 2**: For each level-1 state, generate neighbors
- Hundreds of states explored

**BFS Level 3**: Eventually find [5,4,3,2,1,1]

**Result**: 3 operations (BFS guarantees minimum)

---

## Why This Works for n ≤ 6

**State space size** = n! (all permutations)
- n=2: 2! = 2 states
- n=3: 3! = 6 states
- n=4: 4! = 24 states
- n=5: 5! = 120 states
- n=6: 6! = 720 states

**Transitions per state** ≈ n² subarrays × n insert positions = O(n³)

**Total complexity**: O(n! × n³)
- n=6: 720 × 216 ≈ 155K operations — fast enough!

### Neighbor Generation

```csharp
for (int L = 0; L < n; L++) {
    for (int R = L; R < n; R++) {
        // Extract subarray [L..R]
        var subarray = current[L..R+1];
        
        // Build remaining
        var remaining = current[0..L] + current[R+1..n];
        
        // Try all insert positions
        for (int insertPos = 0; insertPos <= remaining.Count; insertPos++) {
            var newState = remaining[0..insertPos] + subarray + remaining[insertPos..];
            
            if not visited:
                enqueue(newState, ops+1)
        }
    }
}
```

Nested loops: all subarrays × all insert positions.

---

## Why Not Greedy?

**Greedy doesn't work** because local optimal choice may not lead to global optimal.

**Example**: Sometimes moving a "wrong" piece first enables better moves later.

**BFS guarantees**: Explores all paths in order of length, so first solution = shortest.

---

## Key Takeaway

**BFS on permutation state space**. Each state = array configuration. Transitions = split-merge operations. Queue explores shortest paths first. With n ≤ 6, state space (720 max) is small enough for brute-force BFS.

**Think of it as**: Solving a sliding puzzle, but instead of sliding tiles, you're cutting and pasting chunks of array.

## O(n! × n²) — BFS State Space Search

---

## Problem Statement

- Given arrays `nums1` and `nums2` of length `n` (nums2 is permutation of nums1).
- **Operation**: Choose subarray [L..R], remove it, insert anywhere in remaining array.
- Return minimum operations to transform nums1 into nums2.
- **Constraints**: 2 ≤ n ≤ 6, -10⁵ ≤ nums[i] ≤ 10⁵

---

## Core Idea — BFS on State Space

**State**: Current array configuration.

**Transition**: Apply one split-merge operation to get new state.

**Goal**: Find shortest path from nums1 to nums2.

**BFS guarantees**: Minimum number of operations.

---

## Algorithm

```
BFS:
  queue = [(nums1, 0)]  // (state, operations)
  visited = {nums1}
  
  while queue not empty:
    (current, ops) = dequeue()
    
    if current == nums2:
      return ops
    
    // Try all possible split-merge operations
    for L in 0..n-1:
      for R in L..n-1:
        subarray = current[L..R]
        remaining = current[0..L-1] + current[R+1..n-1]
        
        // Try inserting at all positions
        for insertPos in 0..len(remaining):
          newState = remaining[0..insertPos] + subarray + remaining[insertPos+1..]
          
          if newState not in visited:
            visited.add(newState)
            enqueue((newState, ops + 1))
  
  return -1  // Should not happen
```

## Full Implementation (C#)

```csharp
public class Solution {
    public int MinSplitMerge(int[] nums1, int[] nums2) {
        int n = nums1.Length;
        
        // BFS
        var queue = new Queue<(int[], int)>();
        var visited = new HashSet<string>();
        
        string start = ArrayToString(nums1);
        string target = ArrayToString(nums2);
        
        queue.Enqueue((nums1, 0));
        visited.Add(start);
        
        while (queue.Count > 0) {
            var (current, ops) = queue.Dequeue();
            
            string currentStr = ArrayToString(current);
            if (currentStr == target) {
                return ops;
            }
            
            // Try all split-merge operations
            for (int L = 0; L < n; L++) {
                for (int R = L; R < n; R++) {
                    // Extract subarray [L..R]
                    var subarray = new List<int>();
                    for (int i = L; i <= R; i++) {
                        subarray.Add(current[i]);
                    }
                    
                    // Build remaining array
                    var remaining = new List<int>();
                    for (int i = 0; i < L; i++) {
                        remaining.Add(current[i]);
                    }
                    for (int i = R + 1; i < n; i++) {
                        remaining.Add(current[i]);
                    }
                    
                    // Try inserting subarray at all positions
                    for (int insertPos = 0; insertPos <= remaining.Count; insertPos++) {
                        var newState = new List<int>();
                        
                        // Add elements before insert position
                        for (int i = 0; i < insertPos; i++) {
                            newState.Add(remaining[i]);
                        }
                        
                        // Add subarray
                        newState.AddRange(subarray);
                        
                        // Add elements after insert position
                        for (int i = insertPos; i < remaining.Count; i++) {
                            newState.Add(remaining[i]);
                        }
                        
                        string newStateStr = ArrayToString(newState.ToArray());
                        
                        if (!visited.Contains(newStateStr)) {
                            visited.Add(newStateStr);
                            queue.Enqueue((newState.ToArray(), ops + 1));
                        }
                    }
                }
            }
        }
        
        return -1; // Should never reach here
    }
    
    private string ArrayToString(int[] arr) {
        return string.Join(",", arr);
    }
}
```
