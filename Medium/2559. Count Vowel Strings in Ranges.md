# 2559. Count Vowel Strings in Ranges  
*O(n + q) — Prefix Sum of Vowel Strings*

---

## Problem Statement

You are given a **0-indexed** array of strings `words` and a 2D array `queries` where each `queries[i] = [li, ri]`.

For each query, count how many strings in `words[li..ri]` (inclusive) **start** and **end** with a **vowel** ('a', 'e', 'i', 'o', 'u').

Return an array `ans` of size `queries.length` where `ans[i]` is the answer to the i-th query.

**Examples**:

**Example 1**  
Input: words = ["aba","bcb","ece","aa","e"], queries = [[0,2],[1,4],[1,1]]  
Output: **[2,3,0]**  
- [0,2]: "aba", "ece" → 2  
- [1,4]: "ece", "aa", "e" → 3  
- [1,1]: "bcb" → 0

**Example 2**  
Input: words = ["a","e","i"], queries = [[0,2],[0,1],[2,2]]  
Output: **[3,2,1]**  
All strings start & end with vowel

**Constraints**:
- 1 ≤ words.length ≤ 10⁵
- 1 ≤ words[i].length ≤ 40
- sum(words[i].length) ≤ 3×10⁵
- 1 ≤ queries.length ≤ 10⁵
- 0 ≤ li ≤ ri < words.length

---

## Core Idea — Prefix Sum of Vowel-Starting-and-Ending Strings

**Key insight**:
- A string is "good" if it starts **and** ends with a vowel
- We only care about **good strings** in ranges
- Precompute a prefix array where `prefix[i]` = number of good strings in `words[0..i-1]`
- For query [li, ri] → answer = prefix[ri+1] - prefix[li]

**Steps**:
1. Create a boolean array `isGood` where `isGood[i]` = true if `words[i]` starts and ends with vowel
2. Build prefix sum: `prefix[0] = 0`, `prefix[i+1] = prefix[i] + (isGood[i] ? 1 : 0)`
3. For each query [li, ri]: answer = prefix[ri+1] - prefix[li]

**Time**: O(n + q) — one pass to build prefix, one pass for queries  
**Space**: O(n) — prefix array

---

## Clean Implementation (C#)

```csharp
public class Solution {
    public int[] VowelStrings(string[] words, int[][] queries) {
        int n = words.Length;
        
        // Step 1: check which words start and end with vowel
        bool[] isGood = new bool[n];
        char[] vowels = { 'a', 'e', 'i', 'o', 'u' };
        HashSet<char> vowelSet = new HashSet<char>(vowels);
        
        for (int i = 0; i < n; i++) {
            string w = words[i];
            if (w.Length >= 1 && vowelSet.Contains(w[0]) && vowelSet.Contains(w[^1])) {
                isGood[i] = true;
            }
        }
        
        // Step 2: build prefix sum of good strings
        int[] prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + (isGood[i] ? 1 : 0);
        }
        
        // Step 3: answer queries
        int[] ans = new int[queries.Length];
        for (int q = 0; q < queries.Length; q++) {
            int li = queries[q][0];
            int ri = queries[q][1];
            ans[q] = prefix[ri + 1] - prefix[li];
        }
        
        return ans;
    }
}
```

## Complexity

| **Metric**            | **Value**     | **Notes**                                      |
|-----------------------|---------------|------------------------------------------------|
| **Time Complexity**   | **O(n + q)**  | Prefix array computation: O(n)<br>Processing all q queries: O(q) |
| **Space Complexity**  | **O(n)**      | Prefix sum array: O(n)<br>isGood array: O(n)<br>(Can be reduced to O(1) extra space by computing prefix directly in one pass) |

**Optimal** — linear time in total input size, perfect for n, q ≤ 10⁵.

## Why This Works — Example Walkthrough

**Example 1**: words = ["aba", "bcb", "ece", "aa", "e"], queries = [[0,2], [1,4], [1,1]]

**Step 1: Identify good strings** (start **and** end with vowel 'a','e','i','o','u'):

- "aba" → starts 'a', ends 'a' → good
- "bcb" → starts 'b', ends 'b' → not good
- "ece" → starts 'e', ends 'e' → good
- "aa" → starts 'a', ends 'a' → good
- "e" → starts 'e', ends 'e' → good

→ isGood = [true, false, true, true, true]

**Step 2: Build prefix sum** (number of good strings from index 0 to i inclusive):

prefix = [0, 1, 1, 2, 3, 4]

**Step 3: Answer queries** using prefix differences:

- Query [0,2] → prefix[3] - prefix[0] = 2 - 0 = **2** ("aba", "ece")
- Query [1,4] → prefix[5] - prefix[1] = 4 - 1 = **3** ("ece", "aa", "e")
- Query [1,1] → prefix[2] - prefix[1] = 1 - 1 = **0** ("bcb")

Output: **[2,3,0]** → correct

**Correct** —  
We precompute for each word whether it **starts and ends with a vowel** — very simple check on first and last character.  
Then we build a **prefix sum array** that counts how many such good words exist up to each index.  
Every range query [li, ri] becomes a **single subtraction** on the prefix array — O(1) time per query.

---

## Key Takeaway

This is a **classic prefix sum on boolean condition** problem:

* For each word, check if it **starts AND ends** with a vowel → boolean `isGood[i]`
* Build prefix sum array: `prefix[i+1] = prefix[i] + (isGood[i] ? 1 : 0)`
* For each query [li, ri] → answer = `prefix[ri+1] - prefix[li]`
* Vowel check: first char ∈ {'a','e','i','o','u'} **and** last char ∈ {'a','e','i','o','u'}

**Pure, clean, optimal** — O(n + q) time, O(n) space, extremely fast, very simple, and perfectly solves range counting of strings satisfying the vowel start/end property.
