# 1014. Best Sightseeing Pair

---

## Problem Restatement
We are given an array `values` where `values[i]` is the value of the i-th sightseeing spot.  
The score of a pair `(i, j)` with `i < j` is:

```
score(i, j) = values[i] + values[j] + i - j
```
We need to return the maximum possible score.

---

## Key Insight
The formula can be rearranged:
```
values[i] + i + values[j] - j
```

This means:
- For each `j`, we want the maximum of `(values[i] + i)` for all `i < j`.  
- Then add `(values[j] - j)`.

So the problem reduces to maintaining a running maximum of `(values[i] + i)` as we iterate.

---

## Optimized Approach
1. Initialize `best = values[0] + 0`.  
2. Iterate `j` from 1 to n-1:
   - Compute score = `best + values[j] - j`.  
   - Update maxScore.  
   - Update `best = max(best, values[j] + j)`.  

This avoids building a suffix array and runs in **O(n)**.

---

## Code (C#)
```csharp
public class Solution {
    public int MaxScoreSightseeingPair(int[] values) {
        int n = values.Length;
        int best = values[0]; // best candidate for values[i] + i
        int maxScore = int.MinValue;

        for (int j = 1; j < n; j++) {
            maxScore = Math.Max(maxScore, best + values[j] - j);
            best = Math.Max(best, values[j] + j);
        }

        return maxScore;
    }
}
```


## Complexity
- **Time:** O(n), since we only make one pass through the array.  
- **Space:** O(1), using just a few variables (`best`, `maxScore`).  

---

## Example Walkthrough

**Input:** `values = [8,1,5,2,6]`  
- best = 8 (i=0)  
- j=1 → score = 8 + 1 - 1 = 8 → maxScore = 8  
- j=2 → score = 8 + 5 - 2 = 11 → maxScore = 11 ✅  
- j=3 → score = 8 + 2 - 3 = 7  
- j=4 → score = 8 + 6 - 4 = 10  
- **Answer = 11**

**Input:** `values = [1,2]`  
- best = 1  
- j=1 → score = 1 + 2 - 1 = 2 → maxScore = 2 ✅  
- **Answer = 2**

---

## Conclusion
The solution works by **tracking the maximum of (values[i] + i)** as we iterate.  
This allows us to compute the best score in a single pass, yielding an elegant **O(n) time, O(1) space solution**.
