# 3376. Minimum Time to Break Locks I
## O(2^n × n) — Bitmask DP on Lock Order

---

## Problem Statement

- Break n locks in any order. After breaking `bits` locks, factor `x = 1 + bits × k`. Time to break lock with strength `s` at factor `x` = `⌈s/x⌉`. Minimize total time.

---

## Core Idea — Bitmask DP on Subset of Broken Locks

`dp[mask]` = minimum time to break all locks in `mask`.

For each state `mask`, the number of bits set = locks already broken → determines current `x = 1 + popcount(mask) × k`.

For each unbroken lock `i`, try breaking it next: time = `⌈strength[i]/x⌉`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int FindMinimumTime(IList<int> strength, int k) {
        int n = strength.Count;
        int[] dp = new int[1 << n];
        Array.Fill(dp, int.MaxValue);
        dp[0] = 0;

        for (int mask = 0; mask < (1 << n); mask++) {
            if (dp[mask] == int.MaxValue) continue;
            int bits = System.Numerics.BitOperations.PopCount((uint)mask);
            int x = 1 + bits * k;

            for (int i = 0; i < n; i++) {
                if ((mask >> i & 1) == 1) continue;
                int time = (int)Math.Ceiling((double)strength[i] / x);
                int newMask = mask | (1 << i);
                dp[newMask] = Math.Min(dp[newMask], dp[mask] + time);
            }
        }
        return dp[(1<<n)-1];
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(2^n × n)** |
| **Space** | **O(2^n)** |

With n=8: 256 × 8 = 2048 operations — instant.

---

## Key Takeaway

Order matters (x grows with each lock broken) → bitmask DP over all subsets. Time per lock = `⌈strength/x⌉` where `x = 1 + popcount(mask) × k`.
