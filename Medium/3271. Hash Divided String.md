# 3271. Hash Divided String
## O(n) — Direct Simulation

---

## Problem Statement

- Split `s` into blocks of size `k`. For each block: sum char values (a=0..z=25), mod 26, append the corresponding letter to result.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public string StringHash(string s, int k) {
        var sb = new StringBuilder();
        for (int i = 0; i < s.Length; i += k) {
            int sum = 0;
            for (int j = i; j < i+k; j++) sum += s[j]-'a';
            sb.Append((char)('a' + sum % 26));
        }
        return sb.ToString();
    }
}
```

**Time**: O(n) — **Space**: O(n/k) result.

---

## Key Takeaway

Block sum mod 26 → corresponding letter. Straightforward simulation.
