# 3839. Number of Prefix Connected Groups
## O(n · k) — Group by Prefix Substring, Count Groups of Size ≥ 2

---

## Problem Statement

- Two words are prefix-connected if their first `k` characters match.
- A connected group is a set of mutually prefix-connected words.
- Return the number of groups with at least 2 words.
- Words shorter than `k` are ignored.

---

## Core Idea — Prefix as Key in a HashMap

Since all words in a group share the same k-character prefix, just group by `word[0..k-1]`. Count how many groups have size ≥ 2.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int PrefixConnected(string[] words, int k) {
        var groups = new Dictionary<string, int>();

        foreach (string word in words) {
            if (word.Length < k) continue; // ignored
            string prefix = word[..k];
            groups[prefix] = groups.GetValueOrDefault(prefix, 0) + 1;
        }

        int res = 0;
        foreach (int count in groups.Values) {
            if (count >= 2) res++;
        }

        return res;
    }
}
```

---

## Complexity

| Metric | Value | Notes |
|---|---|---|
| Time Complexity | O(n · k) | Substring extraction per word |
| Space Complexity | O(n · k) | Dictionary keys are prefix strings |

## Example Walkthrough

**Example 3:** `words = ["bat","dog","dog","doggy","bat"], k = 3`

| word   | Length ≥ 3? | prefix | count |
|--------|-------------|--------|-------|
| "bat"  | ✓           | "bat"  | 1     |
| "dog"  | ✓           | "dog"  | 1     |
| "dog"  | ✓           | "dog"  | 2     |
| "doggy"| ✓           | "dog"  | 3     |
| "bat"  | ✓           | "bat"  | 2     |

groups: {"bat":2, "dog":3}. Both ≥ 2 → **2** ✓

---

## Pitfalls & Edge Cases

- **Length < k:** skip entirely (word can't share a k-length prefix with anyone). ✓
- **Duplicates count as separate words** — "dog","dog" → same prefix, count = 2 → 1 group. ✓
- **Exactly length k:** `word[..k]` = the entire word. Valid. ✓

---

## Key Takeaway

"Count groups where all members share the same prefix" → group by prefix substring using a HashMap. Trivial once you recognize the equivalence.
