# 515. Find Largest Value in Each Tree Row

---

## Problem Summary
We are given the root of a binary tree.  

For each row (level) of the tree, find the **largest value** among all nodes in that row.  

Return an array of these largest values, ordered by row index (0‑indexed).  

---

## Core Idea
- Perform a **level order traversal (BFS)** using a queue.  
- For each level:  
  - Track the maximum value among all nodes in that level.  
  - Append it to the result.  
- Continue until all levels are processed.  

Alternative: DFS with depth tracking, but BFS is more straightforward.  

---

## C++ Implementation
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector<int> largestValues(TreeNode* root) {
        vector<int> res;
        if (!root) return res;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int size = q.size();
            int maxVal = INT_MIN;
            for (int i = 0; i < size; i++) {
                TreeNode* node = q.front(); q.pop();
                maxVal = max(maxVal, node->val);
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            res.push_back(maxVal);
        }
        return res;
    }
};
```

## Complexity
- **Time:** O(n) — each node is visited exactly once during BFS traversal.  
- **Space:** O(n) — the queue may hold all nodes of a level, worst case up to n/2.  

---

## Pitfalls
- **Empty tree:** return an empty vector immediately.  
- **Negative values:** must initialize `maxVal` with `INT_MIN` to correctly handle negative node values.  
- **Large tree:** up to 10^4 nodes; BFS remains efficient and within constraints.  

---

## Example Walkthrough
**Input:** `root = [1,3,2,5,3,null,9]`  

- **Level 0:** [1] → max = 1  
- **Level 1:** [3,2] → max = 3  
- **Level 2:** [5,3,9] → max = 9  

**Output:** `[1,3,9]`  

---

## Conclusion
The problem reduces to a **level order traversal** with tracking of maximum values per level.  
- BFS provides a clean and efficient solution.  
- Runs in **O(n)** time and **O(n)** space.  
- Robust for trees up to 10^4 nodes, correctly handles negative values and empty trees.
