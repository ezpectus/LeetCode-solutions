# 3365. Rearrange K Substrings to Form Target String
## O(n) — Frequency Map of Substrings

---

## Problem Statement

- Split `s` into `k` substrings of length `n/k`. Can they be rearranged to form `t`?

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public bool IsPossibleToRearrange(string s, string t, int k) {
        int len = s.Length / k;
        var freq = new Dictionary<string, int>();

        for (int i = 0; i < s.Length; i += len) {
            string sub = s.Substring(i, len);
            freq[sub] = freq.GetValueOrDefault(sub) + 1;
        }

        for (int i = 0; i < t.Length; i += len) {
            string sub = t.Substring(i, len);
            if (!freq.TryGetValue(sub, out int cnt) || cnt == 0) return false;
            freq[sub]--;
        }
        return true;
    }
}
```

**Time**: O(n) — k substrings of length n/k, each hashed in O(n/k). Total: O(n).

---

## Key Takeaway

Count substring frequencies from `s`, check they match `t`'s substrings. Straightforward multiset comparison.
