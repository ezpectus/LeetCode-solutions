# 3138. Minimum Length of Anagram Concatenation
## O(n × d(n)) — Try All Divisors, Check Block Frequencies

---

## Problem Statement

- `s` is a concatenation of anagrams of some string `t`.
- Return the minimum possible length of `t`.

---

## Core Idea — Try All Divisors of `len(s)`

The length of `t` must divide `len(s)` (since `s` = `k` copies of anagrams of `t`, so `|s| = k × |t|`). Enumerate all divisors `d` of `n = |s|` from smallest to largest. For each `d`, split `s` into blocks of length `d` and check if all blocks are anagrams of each other (same character frequencies). Return the first valid `d`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinAnagramLength(string s) {
        int n = s.Length;
        for (int d = 1; d <= n; d++) {
            if (n % d != 0) continue;

            // Build frequency of first block
            int[] base_freq = new int[26];
            for (int i = 0; i < d; i++) base_freq[s[i]-'a']++;

            bool valid = true;
            for (int start = d; start < n; start += d) {
                int[] block = new int[26];
                for (int i = start; i < start+d; i++) block[s[i]-'a']++;
                for (int c = 0; c < 26; c++)
                    if (block[c] != base_freq[c]) { valid = false; break; }
                if (!valid) break;
            }

            if (valid) return d;
        }
        return n;
    }
}
```

---

## Complexity

| **Metric** | **Value** | **Notes** |
|---|---|---|
| **Time** | **O(n × d(n))** | d(n) = number of divisors of n; each check is O(n/d × d) = O(n) |
| **Space** | **O(26)** = **O(1)** | Frequency arrays |

`d(n)` is typically very small (~100 for n up to 10⁵) → effectively O(n log n).

---

## Example Walkthrough

**`s="abba"`, n=4**

- d=1: blocks "a","b","b","a". Counter("a")≠Counter("b") → invalid.
- d=2: blocks "ab","ba". Counter("ab")=Counter("ba")={a:1,b:1} ✓ → return **2** ✓

**`s="abcbcacabbaccba"`, n=15**

- d=1: frequencies not all same (a,b,c appear in different blocks) → invalid.
- d=3: blocks "abc","bca","cab","bac","cba". All have {a:1,b:1,c:1} ✓ → return **3** ✓

---

## Pitfalls & Edge Cases

- **`n % d != 0`**: skip non-divisors — blocks wouldn't be equal length.
- **d = n always valid**: `s` itself is an anagram of `t = s` → answer ≤ n. Loop always terminates.
- **Empty substring**: not possible since `d ≥ 1` and blocks are non-empty.

---

## Key Takeaway

"Min length of t such that s = anagram-concatenation of t" → **smallest divisor d of |s| where all d-length blocks have equal character frequencies**:

```
for d in divisors(n) ascending:
    if all blocks s[0..d-1], s[d..2d-1], ... have same freq: return d
```

O(n × number_of_divisors) — efficient for typical inputs.
