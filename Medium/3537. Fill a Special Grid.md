# 3537. Fill a Special Grid
## O(4^n) — Recursive Quadrant Fill

---

## Problem Statement

- Fill a 2ⁿ×2ⁿ grid with 0..4ⁿ-1. Order: top-right < bottom-right < bottom-left < top-left. Each quadrant is also special (recursive).

---

## Core Idea — Recursive Quadrant Assignment

The grid has 4 quadrants. Fill them in order: top-right gets the smallest numbers, then bottom-right, then bottom-left, then top-left (largest). Each quadrant recursively follows the same pattern.

With `q = (size/2)²` numbers per quadrant:
- top-right: `[start, start+q)`
- bottom-right: `[start+q, start+2q)`
- bottom-left: `[start+2q, start+3q)`
- top-left: `[start+3q, start+4q)`

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int[][] SpecialGrid(int n) {
        int size = 1 << n;
        int[][] grid = new int[size][];
        for (int i = 0; i < size; i++) grid[i] = new int[size];
        Fill(grid, 0, 0, size, 0);
        return grid;
    }

    void Fill(int[][] grid, int r, int c, int size, int start) {
        if (size == 1) { grid[r][c] = start; return; }
        int half = size/2, q = half*half;
        Fill(grid, r,      c+half, half, start);      // top-right (smallest)
        Fill(grid, r+half, c+half, half, start+q);    // bottom-right
        Fill(grid, r+half, c,      half, start+2*q);  // bottom-left
        Fill(grid, r,      c,      half, start+3*q);  // top-left (largest)
    }
}
```

**Time**: O(4ⁿ) = O(n²) cells. **Space**: O(4ⁿ) grid + O(n) recursion depth.

---

## Example Walkthrough

**n=1**: size=2, half=1, q=1. start=0.
- top-right (0,1): start=0 → grid[0][1]=0.
- bottom-right (1,1): start=1 → grid[1][1]=1.
- bottom-left (1,0): start=2 → grid[1][0]=2.
- top-left (0,0): start=3 → grid[0][0]=3.

Result: `[[3,0],[2,1]]` ✓

---

## Key Takeaway

Recursive quadrant fill in order (TR, BR, BL, TL) with consecutive ranges. Base case: 1×1 cell gets `start`. Recurse with `size/2` and offset by `(size/2)²` per quadrant.
