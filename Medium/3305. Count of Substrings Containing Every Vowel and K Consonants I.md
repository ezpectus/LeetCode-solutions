# 3305. Count of Substrings Containing Every Vowel and K Consonants I
## O(n²) — Brute Force (n ≤ 250)

---

## Problem Statement

- Count substrings containing all 5 vowels (a,e,i,o,u) at least once and **exactly** k consonants.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int CountOfSubstrings(string word, int k) {
        int n = word.Length, res = 0;
        var vowels = new HashSet<char> {'a','e','i','o','u'};

        for (int i = 0; i < n; i++) {
            int[] freq = new int[26];
            int cons = 0, vowelCount = 0;
            for (int j = i; j < n; j++) {
                char c = word[j];
                if (vowels.Contains(c)) { if (++freq[c-'a'] == 1) vowelCount++; }
                else cons++;
                if (cons > k) break; // can only grow
                if (cons == k && vowelCount == 5) res++;
            }
        }
        return res;
    }
}
```

**Time**: O(n²) — fine for n≤250. Break early when consonants exceed k.

---

## Key Takeaway

Small constraints → brute force all substrings. Early break when `cons > k` keeps it fast.
