# 2825. Make String a Subsequence Using Cyclic Increments
## O(n) — Two Pointers with Cyclic Match

---

## Problem Statement

- Each character in `str1` can be incremented by 1 (cyclically, `'z'→'a'`) at most once.
- Check if `str2` can become a subsequence of the modified `str1`.

---

## Core Idea — Greedy Two Pointers

Standard subsequence check, but with a relaxed match condition: `str1[i]` matches `str2[j]` if:
- `str1[i] == str2[j]` (use as-is), or
- `str1[i] + 1 == str2[j]` cyclically (increment once)

Greedy: scan `str1` left to right with a pointer `j` into `str2`. Advance `j` whenever a match is found. If `j == len(str2)` at the end → true.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public bool CanMakeSubsequence(string str1, string str2) {
        int j = 0;
        foreach (char c in str1) {
            if (j < str2.Length) {
                char t = str2[j];
                char next = (char)((c - 'a' + 1) % 26 + 'a'); // cyclic increment
                if (c == t || next == t) j++;
            }
        }
        return j == str2.Length;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n)** — single pass over str1 |
| **Space** | **O(1)** |

---

## Match Condition

```
next = (c - 'a' + 1) % 26 + 'a'   // 'z' wraps to 'a'
match if: c == t  OR  next == t
```

No need to decide upfront which characters to increment — the greedy matching handles it implicitly.

---

## Example Walkthrough

**`str1="zc", str2="ad"`**

| str1[i] | str2[j] | c==t? | next==t? | match |
|---------|---------|-------|----------|-------|
| z | a | ✗ | z+1='a'==a ✓ | j=1 |
| c | d | ✗ | c+1='d'==d ✓ | j=2 |

`j==2==len(str2)` → **true** ✓

---

## Key Takeaway

"Subsequence check with optional +1 per character" → **standard two-pointer greedy** with extended match condition `c == t || next(c) == t`.
