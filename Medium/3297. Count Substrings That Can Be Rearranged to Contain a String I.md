# 3297. Count Substrings That Can Be Rearranged to Contain a String I
## O(n) — Sliding Window with Deficit Counter

---

## Problem Statement

- Substring of `word1` is valid if it contains all characters of `word2` (with required frequencies).
- Count all valid substrings.

---

## Core Idea — Sliding Window + Count Valid Extensions

Maintain a sliding window `[left, right]`. Track `deficit` = number of distinct characters in `word2` whose required frequency hasn't been met yet.

When `deficit == 0`: window `[left, right]` is valid. Moreover, extending right to `right+1, right+2, ..., n-1` also keeps it valid (more chars can only help). So we add `n - right` to the answer, then shrink from the left to find more minimal valid windows.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public long ValidSubstringCount(string word1, string word2) {
        int[] need = new int[26], freq = new int[26];
        int deficit = 0;
        foreach (char c in word2) if (++need[c-'a'] == 1) deficit++;

        long res = 0;
        int left = 0, n = word1.Length;

        for (int right = 0; right < n; right++) {
            int rc = word1[right]-'a';
            freq[rc]++;
            if (freq[rc] == need[rc]) deficit--;

            while (deficit == 0) {
                res += n - right;
                int lc = word1[left]-'a';
                freq[lc]--;
                if (freq[lc] < need[lc]) deficit++;
                left++;
            }
        }
        return res;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n + m)** — n = word1 length, m = word2 length |
| **Space** | **O(26)** |

## Example Walkthrough

**`word1="abcabc", word2="abc"`** → need: a=1,b=1,c=1, deficit=3.

- right=2 ("abc"): deficit=0. Add n-right=4. Shrink: remove 'a' → deficit=1. left=1.
- right=3 ('a'): deficit=0. Add 3. Shrink: remove 'b' → deficit=1. left=2.
- right=4 ('b'): deficit=0. Add 2. Shrink: remove 'c' → deficit=1. left=3.
- right=5 ('c'): deficit=0. Add 1. Shrink: remove 'a' → deficit=1. left=4.

Total = 4+3+2+1 = **10** ✓

---

## Key Takeaway

"Count substrings containing all of word2" → **sliding window: when deficit==0, add `n-right` (all rightward extensions valid), then shrink left**.
