# 3675. Minimum Operations to Transform String

## O(n) — Find Maximum Character Distance

---

## Problem Statement

- Given string `s` with lowercase letters.
- **Operation**: Replace all occurrences of character `c` with next letter (`c+1`, circular: 'z'→'a').
- Return minimum operations to transform `s` into all 'a' characters.
- **Constraints**: 1 ≤ s.length ≤ 5×10⁵

---

## Core Idea — Maximum Distance to 'a'

**Key observation**: Each character needs a certain number of operations to reach 'a':
- 'a' → 0 operations
- 'b' → 1 operation (b→c→...→z→a = 25 ops, OR b→a = 1 op via reverse)

**Strategy**: The answer is determined by the character **farthest** from 'a'.

So the answer = **maximum distance** among all characters!

**Distance from character `c` to 'a'**:
- 'a' → 0

---

## Algorithm

```
maxDistance = 0

for each char c in s:
  distance = (c == 'a') ? 0 : (26 - (c - 'a'))
  maxDistance = max(maxDistance, distance)

return maxDistance
```

---

## Full Implementation (C#)

```csharp
public class Solution {
    public int MinOperations(string s) {
        int maxOps = 0;
        
        foreach (char c in s) {
            if (c == 'a') continue;
            
            // Distance from c to 'a' going forward (c→c+1→...→z→a)
            int distance = 26 - (c - 'a');
            maxOps = Math.Max(maxOps, distance);
        }
        
        return maxOps;
    }
}
```

## Re-Analysis

**Operation**: Choose character `c`, replace **every** occurrence of `c` with next letter.

**Key**: We choose which character to transform each operation.

**Optimal strategy**: Always transform the character **closest to 'a'** (in circular distance going forward).

## Key Takeaway

**Find character farthest from 'a'** (in forward circular distance). Answer = that maximum distance.

Formula: `distance(c) = (26 - (c - 'a')) % 26` for c ≠ 'a'.

O(n) time, O(1) space.
