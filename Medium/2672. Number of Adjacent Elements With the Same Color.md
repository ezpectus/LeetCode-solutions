# 2672. Number of Adjacent Elements With the Same Color
## O(q) — Incremental Delta on Neighbors

---

## Problem Statement

- Array `colors[n]`, initially all 0 (uncolored).
- Each query sets `colors[idx] = color` and asks: how many adjacent pairs have the same color?

---

## Core Idea — Track Delta, Not Full Recount

Maintain a running count `curr` of same-color adjacent pairs. Each query only affects pairs involving `idx` — at most 2 pairs: `(idx-1, idx)` and `(idx, idx+1)`.

**Per query**:
1. **Before** changing `colors[idx]`: subtract any pairs it was part of (using old value)
2. **Apply** `colors[idx] = color`
3. **After**: add any new pairs formed (using new value)

Only check neighbors if `colors[idx] != 0` (uncolored elements form no pairs).

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int[] ColorTheArray(int n, int[][] queries) {
        int[] colors = new int[n];
        int[] res = new int[queries.Length];
        int curr = 0;

        for (int q = 0; q < queries.Length; q++) {
            int idx = queries[q][0], color = queries[q][1];
            int old = colors[idx];

            // Remove pairs involving idx (old color)
            if (old != 0) {
                if (idx > 0   && colors[idx-1] == old) curr--;
                if (idx < n-1 && colors[idx+1] == old) curr--;
            }

            colors[idx] = color;

            // Add pairs involving idx (new color)
            if (idx > 0   && colors[idx-1] == color) curr++;
            if (idx < n-1 && colors[idx+1] == color) curr++;

            res[q] = curr;
        }

        return res;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(q)** — O(1) per query |
| **Space** | **O(n)** — colors array |

---

## Pitfalls & Edge Cases

- **`old != 0` guard before subtracting** — uncolored cells (0) don't participate in pairs; skip subtraction if cell was uncolored
- **No guard needed when adding** — new color is always ≥ 1, neighbor comparison is safe
- **Same color re-set** — if `color == old`, subtract and re-add cancel out → `curr` unchanged ✓

---

## Key Takeaway

"Maintain global count under point updates" → **delta on affected neighbors only**. Subtract old contributions, apply change, add new contributions. O(1) per update regardless of array size.
