# 3286. Find a Safe Walk Through a Grid
## O(m×n) — 0-1 BFS (Minimum Unsafe Cells on Path)

---

## Problem Statement

- Grid with 0 (safe) and 1 (unsafe, costs 1 health). Start at (0,0), reach (m-1,n-1).
- Return true if minimum health cost of any path < `health`.

---

## Core Idea — 0-1 BFS for Minimum Cost Path

Edge weights are 0 (safe cell) or 1 (unsafe cell). **0-1 BFS** using a deque finds the minimum cost path in O(m×n):
- Cost 0 edge → push to front of deque.
- Cost 1 edge → push to back.

Answer: `dist[m-1][n-1] < health` (need at least 1 health remaining).

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public bool FindSafeWalk(IList<IList<int>> grid, int health) {
        int m = grid.Count, n = grid[0].Count;
        int[,] dist = new int[m, n];
        for (int i = 0; i < m; i++)
            for (int j = 0; j < n; j++) dist[i,j] = int.MaxValue;

        var dq = new LinkedList<(int,int)>();
        dist[0,0] = grid[0][0];
        dq.AddFirst((0, 0));

        int[] dr = {0,0,1,-1}, dc = {1,-1,0,0};
        while (dq.Count > 0) {
            var (r, c) = dq.First.Value; dq.RemoveFirst();
            for (int d = 0; d < 4; d++) {
                int nr = r+dr[d], nc = c+dc[d];
                if (nr < 0 || nr >= m || nc < 0 || nc >= n) continue;
                int nd = dist[r,c] + grid[nr][nc];
                if (nd < dist[nr,nc]) {
                    dist[nr,nc] = nd;
                    if (grid[nr][nc] == 0) dq.AddFirst((nr,nc));
                    else dq.AddLast((nr,nc));
                }
            }
        }
        return dist[m-1,n-1] < health;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(m×n)** |
| **Space** | **O(m×n)** |

---

## Key Takeaway

Binary edge weights (0/1) → 0-1 BFS instead of Dijkstra. Front of deque for free edges, back for cost-1 edges. O(m×n) vs O(m×n log(m×n)) for Dijkstra.
