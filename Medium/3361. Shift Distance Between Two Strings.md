# 3361. Shift Distance Between Two Strings
## O(n + 26) — Prefix Sums for Circular Cost + Per-Char Min

---

## Problem Statement

- Transform `s` into `t` by shifting characters. Each shift step (forward or backward) has a cost per letter. Minimize total cost.

---

## Core Idea — Prefix Sums for O(1) Range Cost

For each position, find the min cost to shift `s[i]` to `t[i]` going either direction on the 26-letter circle.

**Forward cost** (a → a+1 → ... → b): `sum(nextCost[a..b-1])` — use prefix sum on `nextCost`.

**Backward cost** (a → a-1 → ... → b): `sum(previousCost[b+1..a])` — use prefix sum on `previousCost`.

Handle circular wrapping for both directions.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public long ShiftDistance(string s, string t, int[] nextCost, int[] previousCost) {
        long[] nps = new long[27], pps = new long[27];
        for (int i = 0; i < 26; i++) {
            nps[i+1] = nps[i] + nextCost[i];
            pps[i+1] = pps[i] + previousCost[i];
        }

        long FwdCost(int a, int b) =>
            b >= a ? nps[b] - nps[a] : nps[26] - nps[a] + nps[b];

        long BwdCost(int a, int b) =>
            a >= b ? pps[a+1] - pps[b+1]
                   : pps[a+1] + (pps[26] - pps[b+1]);

        long res = 0;
        for (int i = 0; i < s.Length; i++) {
            int a = s[i]-'a', b = t[i]-'a';
            if (a != b)
                res += Math.Min(FwdCost(a, b), BwdCost(a, b));
        }
        return res;
    }
}
```

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n + 26)** = **O(n)** |
| **Space** | **O(1)** |

---

## Cost Formulas

**Forward** `a → b` (going next): 
- If `b >= a`: `sum nextCost[a..b-1]` = `nps[b] - nps[a]`
- If `b < a` (wrap): `nps[26] - nps[a] + nps[b]`

**Backward** `a → b` (going previous):
- If `a >= b`: `sum previousCost[b+1..a]` = `pps[a+1] - pps[b+1]`
- If `a < b` (wrap): `pps[a+1] + (pps[26] - pps[b+1])`

---

## Key Takeaway

Circular alphabet cost queries → prefix sums with wrap handling. O(1) per character pair after O(26) precomputation.
