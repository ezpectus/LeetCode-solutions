# 3775. Reverse Words With Same Vowel Count

## O(n) — Simple String Processing

---

## Problem Statement

- Given string `s` with lowercase words separated by spaces.
- Count vowels in **first word**.
- **Reverse** each following word that has same vowel count.
- Leave other words unchanged.
- **Constraints**: 1 ≤ s.length ≤ 10⁵, lowercase letters + spaces only

---

## Core Idea — Count and Conditionally Reverse

**Algorithm**:
1. Split string into words
2. Count vowels in first word → `targetCount`
3. For each subsequent word:
   - Count vowels
   - If count == targetCount → reverse the word
   - Else → keep unchanged
4. Join words back with spaces

---

## Full Implementation (C#)

```csharp
public class Solution {
    public string ReverseWords(string s) {
        string[] words = s.Split(' ');
        
        // Count vowels in first word
        int targetCount = CountVowels(words[0]);
        
        // Process remaining words
        for (int i = 1; i < words.Length; i++) {
            if (CountVowels(words[i]) == targetCount) {
                words[i] = Reverse(words[i]);
            }
        }
        
        return string.Join(" ", words);
    }
    
    private int CountVowels(string word) {
        int count = 0;
        foreach (char c in word) {
            if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') {
                count++;
            }
        }
        return count;
    }
    
    private string Reverse(string word) {
        char[] arr = word.ToCharArray();
        Array.Reverse(arr);
        return new string(arr);
    }
}
```

## Example Walkthrough

### Example 1: `s = "cat and mice"`

**First word "cat"**: vowels = 'a' → count = 1

**Word "and"**: vowels = 'a' → count = 1 → reverse → "dna"

**Word "mice"**: vowels = 'i', 'e' → count = 2 → unchanged

**Result**: "cat dna mice" ✓

### Example 2: `s = "book is nice"`

**First word "book"**: vowels = 'o', 'o' → count = 2

**Word "is"**: vowels = 'i' → count = 1 → unchanged

**Word "nice"**: vowels = 'i', 'e' → count = 2 → reverse → "ecin"

**Result**: "book is ecin" ✓

---

## Key Takeaway

**Count vowels in first word, conditionally reverse others**. Simple string manipulation. O(n) time and space.
