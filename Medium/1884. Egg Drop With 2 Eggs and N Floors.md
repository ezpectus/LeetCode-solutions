# 1884. Egg Drop With 2 Eggs and N Floors — Architectural Mathematical Closed-Form Solution  
*O(1) — Optimal Direct Formula*

---

## Problem Statement

Given `n` floors and **2 eggs**.

Find **minimum number of drops** in **worst case** to determine exact breaking floor `f` (0 ≤ f ≤ n).

---

## Core Idea — Optimal Drops Form Triangular Numbers

**Key insight**:
- With **2 eggs** and **x drops** in worst case:
  - 1st drop → can afford `x-1` drops left → test `x-1` floors below
  - Total floors covered = `1 + 2 + ... + x = x(x+1)/2`

**Goal**:
- Find **smallest x** such that `x(x+1)/2 ≥ n`

**Solve quadratic**:
```
x(x+1)/2 ≥ n  →  x² + x - 2n ≥ 0
x ≥ (-1 + √(1 + 8n)) / 2
```

Take **ceiling** of that value.

**No loop needed** — direct math.

---

##  Implementation (C#)

```csharp
public class Solution {
    public int TwoEggDrop(int n) {
        int x = 0;
        long sum = 0;
        while (sum < n) {
            x++;
            sum += x;
        }
        
        return x;
    }
}
```

**Correct**.

## Pitfalls & Edge Cases

- **n=1** → 1 drop
- **n=0** → not possible by constraints
- **Large n** → formula handles up to 1000
- **Exact triangular number** → correct x

All handled perfectly.

## Key Takeaway

This is a **beautifully mathematical** closed-form solution:

- **Triangular numbers** → floors covered with x drops
- **Quadratic formula** → minimal x such that x(x+1)/2 ≥ n
- **No simulation** → pure math
**Pure, clean, optimal** — one of the most elegant egg drop variants.
