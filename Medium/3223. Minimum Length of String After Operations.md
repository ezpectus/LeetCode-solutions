# 3223. Minimum Length of String After Operations
## O(n) — Frequency Parity

---

## Problem Statement

- Operation: pick index `i` where the same character exists both to the left and right. Remove the closest left and closest right occurrences.
- Return minimum possible length.

---

## Core Idea — Each Character Reduces to 1 or 2

Each operation on character `c` removes exactly 2 occurrences of `c` (one left, one right of the chosen position). So frequency of `c` decreases by 2 per operation.

We can keep applying operations until:
- Odd frequency → reduces to **1** (can't remove the last one, nothing to pair it with on both sides).
- Even frequency → reduces to **2** (once we have 2 left, they can't be chosen as left+right neighbors simultaneously).

**Answer**: for each distinct character, contribute 1 if odd frequency, 2 if even. Sum all.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinimumLength(string s) {
        int[] freq = new int[26];
        foreach (char c in s) freq[c-'a']++;
        int res = 0;
        foreach (int f in freq)
            if (f > 0) res += f % 2 == 1 ? 1 : 2;
        return res;
    }
}
```

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n)** |

## Example Walkthrough

**`s="abaacbcbb"`**

Frequencies: a=3, b=4, c=2.
- a: odd → 1
- b: even → 2
- c: even → 2

Total = **5** ✓

---

## Key Takeaway

Each operation removes 2 copies of the same character → frequency decreases by 2 each time → parity is preserved. Odd freq → 1 remains, even freq → 2 remain.
