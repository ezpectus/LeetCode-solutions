# 3084. Count Substrings Starting and Ending with Given Character
## O(n) — Count Occurrences, Apply Triangle Number Formula

---

## Problem Statement

- Count substrings of `s` that start AND end with character `c`.
- Single character `c` counts as a valid substring (starts and ends with itself).

---

## Core Idea — Combinatorics on Positions

Let `k` = number of occurrences of `c` in `s`. A valid substring is determined by choosing a **start position** and **end position**, both of which must be positions where `c` occurs, with `start ≤ end`.

- **Pairs with start < end**: `C(k, 2) = k*(k-1)/2` substrings.
- **Single characters (start == end)**: `k` substrings (each occurrence of `c` alone).
- **Total**: `k*(k-1)/2 + k = k*(k+1)/2`.

This is the k-th triangular number.

---

## Full Optimal Implementation (C++)

```cpp
class Solution {
public:
    long long countSubstrings(string s, char c) {
        long long k = count(s.begin(), s.end(), c);
        return k * (k + 1) / 2;
    }
};
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n)** — one pass to count occurrences |
| **Space** | **O(1)** |

---

## Example Walkthrough

**`s="abada", c='a'`**: positions 0, 2, 4 → k=3. Answer = 3×4/2 = **6** ✓

The 6 substrings: "a"(0), "a"(2), "a"(4), "aba"(0-2), "abada"(0-4), "ada"(2-4).

**`s="zzz", c='z'`**: k=3. Answer = 3×4/2 = **6** ✓

---

## Key Takeaway

"Count substrings starting and ending with c" → **count occurrences k, return k*(k+1)/2**. Every pair of c-positions (including same position) defines exactly one valid substring.
