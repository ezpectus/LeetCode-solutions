# 2707. Extra Characters in a String
## O(n² × L) — DP on String Prefixes

---

## Problem Statement

- Split `s` into non-overlapping substrings from `dictionary`, minimizing leftover characters.
- Return the minimum number of extra (unused) characters.

---

## Core Idea — DP on Prefix Lengths

**State**: `dp[i]` = minimum extra characters in `s[0..i-1]` (first `i` characters).

**Transition**: for each position `i`, two choices:
1. **`s[i-1]` is extra** → `dp[i] = dp[i-1] + 1`
2. **`s[j..i-1]` matches a dictionary word** → `dp[i] = min(dp[i], dp[j])` for all `j < i`

The second option says: if the substring ending at `i` is a valid word, the extra characters up to `i` = extra characters up to `j` (no new extras from the matched word).

**Base case**: `dp[0] = 0` (empty prefix, 0 extra characters).

**Answer**: `dp[n]`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MinExtraChar(string s, string[] dictionary) {
        int n = s.Length;
        var words = new HashSet<string>(dictionary);
        int[] dp = new int[n + 1];

        for (int i = 1; i <= n; i++) {
            dp[i] = dp[i-1] + 1; // s[i-1] is extra
            for (int j = 0; j < i; j++) {
                if (words.Contains(s.Substring(j, i - j)))
                    dp[i] = Math.Min(dp[i], dp[j]);
            }
        }

        return dp[n];
    }
}
```

---

## Complexity

| **Metric** | **Value** | **Notes** |
|---|---|---|
| **Time Complexity** | **O(n² × L)** | n² substrings × O(L) hash lookup (L = avg word length) |
| **Space Complexity** | **O(n + W)** | dp array + HashSet of dictionary words |

With n ≤ 50 and dictionary ≤ 50 words, this is trivially fast.

## Example Walkthrough

**Example 1**: `s = "leetscode"`, `dictionary = ["leet","code","leetcode"]`

| i | dp[i-1]+1 | matching words ending at i | dp[i] |
|---|-----------|---------------------------|-------|
| 1 | 1 | none | 1 |
| 2 | 2 | none | 2 |
| 3 | 3 | none | 3 |
| 4 | 4 | "leet" (j=0) → dp[0]=0 | **0** |
| 5 | 1 | none | 1 |
| 6 | 2 | none | 2 |
| 7 | 3 | none | 3 |
| 8 | 4 | none | 4 |
| 9 | 5 | "code" (j=5) → dp[5]=1 | **1** |

`dp[9] = 1` ✓

---

## Pitfalls & Edge Cases

- **No matching word at all** — `dp[i] = dp[i-1] + 1` always fires → result = `n` (all characters extra) ✓
- **Overlapping words** — DP naturally handles this; each `j` is tried independently
- **Whole string is a word** — `j=0, i=n` → `dp[n] = dp[0] = 0` ✓
- **`s.Substring` in C#** — `s.Substring(j, i-j)` takes start and **length**, not end index

---

## Key Takeaway

"Minimum leftover characters when splitting with dictionary" → **prefix DP**:

```
dp[i] = min(
    dp[i-1] + 1,                  // current char is extra
    dp[j] for all j where s[j..i-1] ∈ dictionary  // match a word
)
```

Same pattern as Word Break but minimizing extras instead of checking possibility.
