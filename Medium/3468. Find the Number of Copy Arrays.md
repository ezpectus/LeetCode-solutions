# 3468. Find the Number of Copy Arrays
## O(n) — Intersect Constraints on First Element

---

## Problem Statement

- `copy` must have same differences as `original`, and `bounds[i][0] ≤ copy[i] ≤ bounds[i][1]`.
- Count valid arrays.

---

## Core Idea — Everything Determined by copy[0]

Since differences are fixed: `copy[i] = copy[0] + (original[i] - original[0])`.

For each `i`, the constraint `bounds[i][0] ≤ copy[i] ≤ bounds[i][1]` translates to:
```
bounds[i][0] - diff[i] ≤ copy[0] ≤ bounds[i][1] - diff[i]
```
where `diff[i] = original[i] - original[0]`.

Intersect all these ranges for `copy[0]`. Answer = max(0, hi - lo + 1).

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int CountArrays(int[] original, int[][] bounds) {
        long lo = bounds[0][0], hi = bounds[0][1];
        for (int i = 1; i < original.Length; i++) {
            long diff = original[i] - original[0];
            lo = Math.Max(lo, bounds[i][0] - diff);
            hi = Math.Min(hi, bounds[i][1] - diff);
        }
        return (int)Math.Max(0, hi - lo + 1);
    }
}
```

**Time**: O(n) — **Space**: O(1).

---

## Example Walkthrough

**`original=[1,2,3,4], bounds=[[1,2],[2,3],[3,4],[4,5]]`**

diff = [0,1,2,3]. Constraints on copy[0]:
- i=0: [1,2]
- i=1: [2-1,3-1]=[1,2]
- i=2: [3-2,4-2]=[1,2]
- i=3: [4-3,5-3]=[1,2]

Intersection: [1,2]. Count = **2** ✓

---

## Key Takeaway

Fixed differences → array determined by first element → intersect per-element bounds mapped to copy[0] range → O(n).
