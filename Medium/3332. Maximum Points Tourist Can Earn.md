# 3332. Maximum Points Tourist Can Earn
## O(k × n²) — DP on (Day, City)

---

## Problem Statement

- k days, n cities. Each day: stay (earn `stayScore[day][city]`) or travel to another city (earn `travelScore[city][dest]`). Start anywhere. Maximize total points.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int MaxScore(int n, int k, int[][] stayScore, int[][] travelScore) {
        int[,] dp = new int[k+1, n]; // dp[day][city] = max points at start of day

        for (int day = 0; day < k; day++)
            for (int city = 0; city < n; city++) {
                // Stay
                dp[day+1, city] = Math.Max(dp[day+1, city],
                    dp[day, city] + stayScore[day][city]);
                // Travel
                for (int dest = 0; dest < n; dest++)
                    if (dest != city)
                        dp[day+1, dest] = Math.Max(dp[day+1, dest],
                            dp[day, city] + travelScore[city][dest]);
            }

        int res = 0;
        for (int c = 0; c < n; c++) res = Math.Max(res, dp[k, c]);
        return res;
    }
}
```

**Time**: O(k × n²) — **Space**: O(k × n), reducible to O(n) with rolling array.

---

## Key Takeaway

Standard DP: `dp[day+1][city] = max(stay, travel from any city)`. Initialize all zeros (can start at any city for free).
