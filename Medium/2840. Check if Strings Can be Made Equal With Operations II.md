# 2840. Check if Strings Can be Made Equal With Operations II
## O(n) — Even/Odd Positions Are Independent

---

## Problem Statement

- Swap characters at indices `i` and `j` if `j - i` is even (same parity).
- Return true if `s1` can be made equal to `s2`.

---

## Core Idea — Parity Partitioning

Swapping indices with even difference means **even indices only swap among even indices, odd only among odd**. Within each parity group, any arrangement is reachable (any permutation via transpositions). So `s1` can become `s2` iff:

- `sorted(s1[even positions]) == sorted(s2[even positions])`
- `sorted(s1[odd positions]) == sorted(s2[odd positions])`

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public bool CheckStrings(string s1, string s2) {
        int[] even = new int[26], odd = new int[26];
        for (int i = 0; i < s1.Length; i++) {
            if (i % 2 == 0) { even[s1[i]-'a']++; even[s2[i]-'a']--; }
            else             { odd[s1[i]-'a']++;  odd[s2[i]-'a']--;  }
        }
        return even.All(x => x == 0) && odd.All(x => x == 0);
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n)** |
| **Space** | **O(1)** — fixed 26-char arrays |

---

## Key Takeaway

"Swap with even index difference" → even and odd positions are independent permutation groups. Check frequency equality within each group.
