- Use **edge-guided traversal** starting from the top-right corner  
- Move **left** if current value is too big  
- Move **down** if current value is too small  
- Stop if match is found  
- Avoid extra space â€” no HashSet or auxiliary structures

## ðŸ§© Solution â€” Top-Right Search C#

```csharp
public class Solution {
    public bool SearchMatrix(int[][] matrix, int target) {
        int row = 0;
        int col = matrix[0].Length - 1;

        while (row < matrix.Length && col >= 0) {
            if (matrix[row][col] == target) {
                return true;
            } else if (matrix[row][col] > target) {
                col--;
            } else {
                row++;
            }
        }

        return false;
    }
}
```

### ðŸ”— Why Top-Right?

- From the **top-right corner**, you can eliminate one row or one column at each step  
- Matrix is sorted â†’ guarantees **directional movement**  
- No need for binary search or full traversal â€” just guided descent
