# 3558. Number of Ways to Assign Edge Weights I  
## O(n) — Depth + Parity DP on a Single Path

---

## Problem Summary

You are given a **tree** rooted at node `1`.  
You must:

1. Pick **any node at maximum depth** (deepest node).
2. Consider the **unique path** from root `1` to that node.
3. Each edge on that path must be assigned weight **1 or 2**.
4. Count the number of assignments such that the **total path cost is odd**.
5. Ignore all edges not on that path.

Return the count modulo `1e9+7`.

---

## Key Insight

Let the depth of the chosen deepest node be:

```
L = number of edges from root to that node
```

Each edge weight is either:

- `1` (odd)
- `2` (even)

We need:

```
sum(weights) % 2 == 1
```

Since:

```
2 % 2 = 0
1 % 2 = 1
```

The parity of the total sum is simply:

```
(# of edges assigned weight 1) % 2
```

Thus:

> We need an **odd number of edges** assigned weight `1`.

Total edges on the path = `L`.

Each edge independently chooses weight `{1,2}` → total assignments:

```
2^L
```

Number of assignments with **odd number of 1's**:

```
2^(L-1)
```

This is a classic combinatorial identity:

- Number of subsets of size **odd** = `2^(L-1)`
- Number of subsets of size **even** = `2^(L-1)`

---

## Steps to Solve

### 1. Build adjacency list  
Tree is undirected.

### 2. BFS/DFS from root to compute depths  
Track:

- `maxDepth`
- any node achieving that depth

### 3. Compute `L = maxDepth`  
Because depth = number of edges from root.

### 4. Answer = `2^(L-1) mod MOD`

---

## Example

### Example 2

Tree:

```
1
├─ 2
└─ 3
   ├─ 4
   └─ 5
```

Deepest nodes: `4` and `5`, depth = 2 → path length `L = 2`.

Number of valid assignments:

```
2^(2-1) = 2
```

Matches sample.

## Full C++ Implementation

```cpp
class Solution {
public:
    static const long long MOD = 1000000007;

    long long modpow(long long a, long long e) {
        long long r = 1;
        while (e > 0) {
            if (e & 1) r = (r * a) % MOD;
            a = (a * a) % MOD;
            e >>= 1;
        }
        return r;
    }

    int assignEdgeWeights(vector<vector<int>>& edges) {
        int n = edges.size() + 1;

        vector<vector<int>> g(n + 1);
        for (auto &e : edges) {
            int u = e[0], v = e[1];
            g[u].push_back(v);
            g[v].push_back(u);
        }

        // BFS from root = 1
        vector<int> depth(n + 1, -1);
        queue<int> q;
        depth[1] = 0;
        q.push(1);

        int maxDepth = 0;

        while (!q.empty()) {
            int u = q.front(); q.pop();
            for (int v : g[u]) {
                if (depth[v] == -1) {
                    depth[v] = depth[u] + 1;
                    maxDepth = max(maxDepth, depth[v]);
                    q.push(v);
                }
            }
        }

        // Path length = maxDepth
        int L = maxDepth;

        if (L == 0) return 0; // only root exists

        // Number of assignments with odd sum = 2^(L-1)
        return (int)modpow(2, L - 1);
    }
};
```

---

## Complexity

| Step | Cost |
|------|------|
| BFS | O(n) |
| Final computation | O(log MOD) |
| Total | **O(n)** |

---

## Key Takeaway

The entire problem reduces to:

> Count assignments of `{1,2}` on a path of length `L` such that the sum is odd → `2^(L-1)`.

Everything else is just finding the deepest node.
