# 3301. Maximize the Total Height of Unique Towers
## O(n log n) — Sort Descending + Greedy

---

## Problem Statement

- Assign distinct positive integer heights to towers, each ≤ `maximumHeight[i]`. Maximize total sum.

---

## Core Idea — Sort Descending, Assign Greedily

Sort towers by max height descending. Assign the first tower its maximum height. For each subsequent tower, assign `min(maximumHeight[i], prev - 1)` — as large as possible while still being less than the previous assignment.

If at any point the assigned height ≤ 0: impossible, return -1.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public long MaximumTotalSum(int[] maximumHeight) {
        var sorted = maximumHeight.OrderByDescending(x => x).ToArray();
        long res = sorted[0];
        int prev = sorted[0];

        for (int i = 1; i < sorted.Length; i++) {
            int h = Math.Min(sorted[i], prev-1);
            if (h <= 0) return -1;
            res += h;
            prev = h;
        }
        return res;
    }
}
```

| **Metric** | **Value** |
|---|---|
| **Time** | **O(n log n)** |

## Example Walkthrough

**`maximumHeight=[2,3,4,3]`** → sorted: [4,3,3,2].
- h=4, h=min(3,3)=3, h=min(3,2)=2, h=min(2,1)=1. Total=4+3+2+1=**10** ✓

---

## Key Takeaway

Sort descending → greedily assign `min(max_h, prev-1)`. Each tower gets maximum possible unique height below the previous.
