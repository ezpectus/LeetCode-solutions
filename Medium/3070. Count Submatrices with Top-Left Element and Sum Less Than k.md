# 3070. Count Submatrices with Top-Left Element and Sum ≤ k
## O(m × n) — 2D Prefix Sum

---

## Problem Statement

- Count submatrices that include cell `(0,0)` and have sum ≤ `k`.
- All such submatrices are defined by their bottom-right corner `(r, c)`.

---

## Core Idea — 2D Prefix Sum

Every submatrix containing `(0,0)` is uniquely identified by its bottom-right corner `(r,c)`. Its sum = `pref[r][c]` = sum of the rectangle `[0,0]..[r,c]`.

Build the 2D prefix sum in-place:

```
pref[r][c] = grid[r][c]
           + pref[r-1][c]    (above)
           + pref[r][c-1]    (left)
           - pref[r-1][c-1]  (subtract double-counted corner)
```

Count cells where `pref[r][c] ≤ k`.

---

## Full Optimal Implementation (C#)

```csharp
public class Solution {
    public int CountSubmatrices(int[][] grid, int k) {
        int m = grid.Length, n = grid[0].Length, res = 0;
        int[,] pref = new int[m, n];

        for (int r = 0; r < m; r++)
            for (int c = 0; c < n; c++) {
                pref[r, c] = grid[r][c];
                if (r > 0) pref[r, c] += pref[r-1, c];
                if (c > 0) pref[r, c] += pref[r, c-1];
                if (r > 0 && c > 0) pref[r, c] -= pref[r-1, c-1];
                if (pref[r, c] <= k) res++;
            }

        return res;
    }
}
```

---

## Complexity

| **Metric** | **Value** |
|---|---|
| **Time** | **O(m × n)** |
| **Space** | **O(m × n)** — prefix sum array |

---

## 2D Prefix Sum Formula

```
pref[r][c] = grid[r][c]
           + pref[r-1][c]   ← column above
           + pref[r][c-1]   ← row to left
           - pref[r-1][c-1] ← top-left corner added twice, subtract once
```

This is the standard inclusion-exclusion for 2D prefix sums.

---

## Pitfalls & Edge Cases

- **Bounds checks** — `r > 0` and `c > 0` before accessing previous cells
- **`grid[0][0] > k`** — the single-cell submatrix already exceeds `k`; res stays 0 ✓

---

## Key Takeaway

"Count top-left submatrices with sum ≤ k" → **2D prefix sum**: each bottom-right corner `(r,c)` defines exactly one such submatrix. Build prefix sum on the fly, count cells ≤ k.
